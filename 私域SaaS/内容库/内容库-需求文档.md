# 内容库需求文档

## 1. 需求概述

内容库用于沉淀可分发给用户的推广图文素材，并支持在积分任务中关联启用状态的内容库。用户参与积分任务领取推广素材时，系统按内容领取状态记录素材是否已被领取；运营可在后台维护内容库、管理内容、导入内容、查看领取用户记录。

本需求覆盖：

- 内容库列表
- 内容库新增/编辑
- 内容管理列表
- 内容新增/编辑
- 内容导入
- 领取用户记录
- 积分任务新增/编辑中的任务素材配置

## 2. 业务规则

1. 内容库创建后默认开启，状态为“启用”。
2. 只有“启用”状态的内容库可被积分任务选择关联。
3. 内容库可直接在列表中启用或禁用。
4. 禁用内容库后，该内容库不再出现在积分任务的内容库选择弹窗中。
5. 一个积分任务一次只能选择一个内容库。
6. 内容库下的内容领取状态仅包含“未领取”“已领取”。
7. 有用户通过积分任务成功领取到内容后，该内容记录为“已领取”。
8. 未被用户成功领取的内容记录为“未领取”。
9. 内容管理新增/编辑时不允许人工维护领取状态。
10. 内容正文在同一内容库下不可重复。
11. 点击“领取用户”时，仅成功通过活动领取该内容的用户有数据展示；无成功领取记录时展示空列表。

## P101 内容库列表

# 页面路径

内容中心 - 内容库

---

# 流程状态

| 状态 | 说明 | 后续操作 |
|---|---|---|
| 启用 | 内容库可被积分任务选择关联 | 可禁用、编辑、内容管理、删除 |
| 禁用 | 内容库不可被积分任务选择关联 | 可启用、编辑、内容管理、删除 |

---

# 列表逻辑

## 表搜索

| 字段 | 类型 | 说明 |
|---|---|---|
| 内容库名称 | 文本输入框 | 支持按内容库名称模糊搜索 |
| 状态 | 下拉选择 | 全部、启用、禁用 |
| 创建人 | 文本输入框 | 支持按创建人模糊搜索 |
| 搜索 | 按钮 | 点击后按当前筛选条件刷新列表 |
| 清空 | 文本按钮 | 清空筛选条件并刷新列表 |

## 查询表

| 字段 | 说明 |
|---|---|
| 内容库信息 | 展示内容库名称、ID；名称可点击进入内容管理 |
| 内容数据 | 展示内容总数、可分发内容数、已领取内容数 |
| 创建时间 | 展示创建人、创建时间 |
| 更新时间 | 展示更新人、更新时间 |
| 状态 | 开关控件；开启表示启用，关闭表示禁用 |
| 操作 | 编辑、内容管理、删除 |

## 分页

| 字段 | 说明 |
|---|---|
| 总条数 | 展示当前查询结果总数 |
| 页码 | 支持切换页码 |
| 每页条数 | 默认 15 条/页 |

---

# 列表交互

| 触发点 | 系统响应 |
|---|---|
| 点击“新增” | 进入新建内容库页面 |
| 点击内容库名称 | 进入该内容库的内容管理列表 |
| 点击“编辑” | 进入编辑内容库页面 |
| 点击“内容管理” | 进入该内容库的内容管理列表 |
| 点击状态开关 | 切换启用/禁用状态，并更新更新时间、更新人 |
| 点击“删除” | 弹出删除确认；确认后删除该内容库 |

## 删除规则

1. 删除前必须二次确认。
2. 删除后该内容库不再展示在列表中。
3. 删除内容库后，该内容库下内容同步移除。
4. 如内容库已被任务关联，删除时需同步解除任务关联关系。

## 状态切换规则

1. 启用后，内容库可在积分任务新增/编辑中被选择。
2. 禁用后，内容库不再出现在积分任务的内容库选择弹窗中。
3. 状态切换不改变内容库下已有内容的领取状态。

## P102 内容库新增/编辑

# 页面路径

内容中心 - 内容库 - 新建内容库 / 编辑内容库

---

# 表单逻辑

## 基本信息

| 字段 | 类型 | 必填 | 规则 |
|---|---|---|---|
| 内容库名 | 文本输入框 | 是 | 50 字内；名称不能与已有内容库重复 |

## 默认状态

| 字段 | 规则 |
|---|---|
| 状态 | 新建内容库默认启用 |

---

# 列表交互

| 触发点 | 系统响应 |
|---|---|
| 点击“保存” | 校验必填项、字数、名称唯一性；校验通过后保存 |
| 点击“取消” | 返回内容库列表，不保存当前修改 |
| 点击返回图标 | 返回内容库列表，不保存当前修改 |

## 校验规则

1. 内容库名为空时，不允许保存。
2. 内容库名超过 50 字时，不允许保存。
3. 内容库名与已有内容库重复时，不允许保存。
4. 编辑内容库时，允许保留原内容库名称。

1. 仅用户通过活动成功领取内容时，弹窗展示对应领取记录。
2. 未领取内容点击“领取用户”时，弹窗展示空列表。
3. 已领取内容如无成功领取记录，弹窗展示空列表。

## P104 内容新增/编辑

# 页面路径

内容中心 - 内容库 - 内容管理 - 新建内容 / 编辑内容

---

# 表单逻辑

## 基础内容

| 字段 | 类型 | 必填 | 规则 |
|---|---|---|---|
| 标题 | 文本输入框 | 是 | 200 字内；支持表情、符号 |
| 正文 | 多行文本框 | 是 | 800 字内；支持换行、表情、符号；同一内容库下正文不可重复 |
| 标签 | 文本输入框 | 否 | 用于展示内容标签 |

## 图片素材

| 字段 | 类型 | 必填 | 规则 |
|---|---|---|---|
| 封面图 | 图片上传 | 否 | 最多 2 张 |
| 细节图 | 图片上传 | 否 | 最多 20 张 |

---

# 列表交互

| 触发点 | 系统响应 |
|---|---|
| 点击“保存” | 校验必填项、字数、正文唯一性；校验通过后保存 |
| 点击“取消” | 返回内容管理列表，不保存当前修改 |
| 点击返回图标 | 返回内容管理列表，不保存当前修改 |
| 点击上传图片 | 选择并上传图片 |
| 点击删除图片 | 从当前内容中移除该图片 |

## 校验规则

1. 标题为空时，不允许保存。
2. 正文为空时，不允许保存。
3. 标题超过 200 字时，不允许保存。
4. 正文超过 800 字时，不允许保存。
5. 同一内容库下正文重复时，不允许保存。
6. 新增内容保存后，领取状态默认为未领取。
7. 领取状态由用户领取行为自动产生，不在表单中展示和编辑。

# 列表交互

| 触发点 | 系统响应 |
|---|---|
| 点击“内容导入” | 打开内容导入弹窗 |

| 点击“取消” | 关闭弹窗，不导入 |

1. 导入内容归属当前内容库。

4. 选择“覆盖已有内容”时，系统使用导入内容覆盖重复正文对应的已有内容。

## P106 积分任务新增/编辑 - 任务素材

# 页面路径

积分中心 - 积分任务 - 新建任务 / 编辑任务

---

# 表单逻辑

## 任务素材

| 字段 | 类型 | 必填 | 规则 |
|---|---|---|---|
| 内容库 | 选择器 | 否 | 只能选择启用状态的内容库；一次只能选择 1 个 |
| 默认素材 | 复选框 | 否 | 勾选后启用默认素材 |
| 标题 | 文本输入框 | 启用默认素材时必填 | 200 字内；支持表情、符号 |
| 正文 | 多行文本框 | 启用默认素材时必填 | 800 字内；支持表情、符号 |
| 标签 | 文本输入框 | 否 | 用于默认素材标签 |
| 封面图 | 图片上传 | 否 | 最多 10 张 |
| 细节图 | 图片上传 | 否 | 最多 10 张 |


---

# 列表交互

| 触发点 | 系统响应 |
|---|---|
| 点击内容库选择 | 打开内容库选择弹窗 |
| 选择内容库 | 记录当前任务关联的内容库 |
| 勾选默认素材 | 展开默认素材填写区域 |
| 取消勾选默认素材 | 收起默认素材填写区域，并按规则不分发默认素材 |

## 内容库选择规则

1. 内容库选择弹窗仅展示启用状态的内容库。
2. 一个任务一次只能选择一个内容库。
3. 禁用状态内容库不可被新任务或编辑任务选择。
4. 内容库被选择后，任务领取素材时优先从该内容库中分发未领取内容。

## 默认素材分发规则

1. 启用默认素材时，当领取素材用户数大于内容库内容条数，系统统一分发默认素材内容。
2. 未启用默认素材时，当领取素材用户数大于内容库内容条数，系统不再分发默认素材。
3. 默认素材仅作为内容库不足时的兜底素材，不写入内容库内容列表。

## 保存校验

1. 内容库、默认素材必填一项
2. 启用默认素材时，默认素材标题、正文必填。
3. 未选择内容库且启用默认素材时，可保存任务，但素材分发仅使用默认素材。
