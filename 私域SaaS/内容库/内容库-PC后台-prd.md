# 页面路径

内容中心 - 内容库

---

# 流程状态

> 说明：本页面存在内容库启用状态流转，各状态下可执行操作如下。

1. **启用**：可执行编辑、内容管理、禁用、删除；该内容库可在积分任务新增/编辑中被选择关联。
2. **禁用**：可执行编辑、内容管理、启用、删除；该内容库不可在积分任务新增/编辑中被选择关联。
3. **启用 -> 禁用**：运营点击状态开关关闭后，内容库状态变为禁用，列表刷新状态开关，并记录更新人、更新时间。
4. **禁用 -> 启用**：运营点击状态开关开启后，内容库状态变为启用，列表刷新状态开关，并记录更新人、更新时间。

---

# 列表逻辑

## 表搜索

1. 内容库名称：文本输入-模糊匹配，匹配内容库名称字段，支持空值查询；为空时不参与筛选。
2. 状态：下拉单选，枚举值为全部、启用、禁用，默认全部；选择启用时仅查询启用内容库，选择禁用时仅查询禁用内容库。
3. 创建人：文本输入-模糊匹配，匹配内容库创建人字段，支持空值查询；为空时不参与筛选。
4. 搜索按钮：点击后按当前筛选条件查询内容库列表。
5. 清空：点击后清空内容库名称、状态、创建人筛选条件，并刷新列表。
6. 多字段间搜索关系：AND，同时满足全部非空筛选条件。

## 查询表

**排序与分页**：默认按更新时间倒序排列，默认 15 条/每页，支持分页切换。

1. 内容库信息
   - 1.1 内容库名称：取内容库名称字段，蓝色文字展示；点击进入该内容库的内容管理列表。
   - 1.2 内容库 ID：取内容库 ID 字段，展示格式为“ID：10004”。
   - 1.3 空值处理：内容库名称和 ID 为必有字段，不展示空值。

2. 内容数据
   - 2.1 内容总数：统计当前内容库下未删除内容条数，展示格式为“共 N 条”。
   - 2.2 可分发内容：统计当前内容库下领取状态为未领取的内容条数，展示格式为“可分发 N”。
   - 2.3 已领取内容：统计当前内容库下领取状态为已领取的内容条数，展示格式为“已领取 N”。

3. 创建时间
   - 3.1 创建人：取内容库创建人字段，第一行展示。
   - 3.2 创建时间：取内容库创建时间字段，第二行展示，时间格式为 `YYYY-MM-DD HH:mm:ss`，时区为系统后台时区。

4. 更新时间
   - 4.1 更新人：取内容库最近更新人字段，第一行展示。
   - 4.2 更新时间：取内容库最近更新时间字段，第二行展示，时间格式为 `YYYY-MM-DD HH:mm:ss`，时区为系统后台时区。

5. 状态
   - 5.1 启用：开关为开启态，使用主色展示。
   - 5.2 禁用：开关为关闭态，使用灰色展示。
   - 5.3 状态字段不展示文字标签，仅通过开关状态表达。

6. 操作列
   - 6.1 启用状态展示：编辑、内容管理、删除。
   - 6.2 禁用状态展示：编辑、内容管理、删除。
   - 6.3 操作入口使用文字链接展示，删除使用红色文字。

# 列表交互

1. **新增**：跳转页面。
   - 1.1 点击「新增」按钮，跳转至内容中心 - 内容库 - 新建内容库。
   - 1.2 新建保存成功后返回内容库列表，并刷新列表数据。

2. **内容库名称**：跳转页面。
   - 2.1 点击内容库名称，跳转至内容中心 - 内容库 - 内容管理。
   - 2.2 进入内容管理后，仅展示当前内容库下的内容数据。

3. **编辑**：跳转页面。
   - 3.1 点击「编辑」文字链接，跳转至内容中心 - 内容库 - 编辑内容库。
   - 3.2 编辑保存成功后返回内容库列表，并刷新列表数据。

4. **内容管理**：跳转页面。
   - 4.1 点击「内容管理」文字链接，跳转至内容中心 - 内容库 - 内容管理。
   - 4.2 页面携带当前内容库 ID 作为查询条件。

5. **状态开关**：状态切换。
   - 5.1 启用状态点击开关，状态变更为禁用。
   - 5.2 禁用状态点击开关，状态变更为启用。
   - 5.3 状态切换成功后 Toast 提示「状态已更新」。
   - 5.4 状态切换成功后更新最近更新人、最近更新时间。
   - 5.5 接口失败时 Toast 提示接口返回的错误信息，开关恢复切换前状态。

6. **删除**：展示提示框。
   - 6.1 点击「删除」文字链接，展示二次确认提示框。
   - 6.2 确认文案：「是否确认删除」。
   - 6.3 点击「取消」关闭提示框，不删除数据。
   - 6.4 点击「确定」后删除内容库，删除成功 Toast 提示「删除成功」。
   - 6.5 删除成功后，列表移除该内容库，并同步移除该内容库下内容。
   - 6.6 待确认：若内容库已被积分任务关联，删除时是否需要阻断删除或仅解除关联。

# 表单逻辑

## 基本信息

1. 内容库名：文本输入，必填，最多 50 字；用于创建或编辑内容库名称。
   - 1.1 新增时，内容库名默认为空。
   - 1.2 编辑时，内容库名回显当前内容库名称。

   - 1.5 内容库名超过 50 字时不允许保存。

2. 状态：系统默认字段，不在表单中展示。
   - 2.1 新建内容库保存成功后，状态默认启用。
   - 2.2 编辑内容库不修改状态。
   - 2.3 状态仅在内容库列表中通过开关维护。

# 列表交互

1. **保存**：点击「保存」按钮。
   - 1.1 触发表单校验，校验内容包含内容库名必填、长度、唯一性。
   - 1.2 内容库名为空时，对应字段下方展示错误提示文案：「请输入内容库名」。
   - 1.3 内容库名重复时，对应字段下方展示错误提示文案：「名称不能与已有内容库重复」。
   - 1.4 校验通过后调用保存接口，成功后 Toast 提示「保存成功」，跳转回内容库列表。
   - 1.5 接口失败时 Toast 提示接口返回的错误信息。

2. **取消**：点击「取消」按钮。
   - 2.1 如表单无修改，直接返回内容库列表。
   - 2.2 如表单有未保存内容，展示二次确认：「确认离开？当前修改将不会保存」。
   - 2.3 确认后返回内容库列表，取消后留在当前页面。

3. **返回**：点击页面左上角返回图标。
   - 3.1 交互逻辑同取消。

# 流程状态

> 说明：本页面存在内容领取状态，各状态下可执行操作如下。

1. **未领取**：可执行编辑、领取用户、删除；表示该内容尚未被用户通过积分任务成功领取。
2. **已领取**：可执行编辑、领取用户、删除；表示该内容已被用户通过积分任务成功领取。
3. **未领取 -> 已领取**：用户通过积分任务成功领取该内容后，系统自动将领取状态更新为已领取，并记录领取用户、活动信息、领取时间。
4. **已领取 -> 未领取**：后台不提供人工回退入口。

   - 3.3 展示格式为 `YYYY-MM-DD HH:mm:ss`，时区为系统后台时区。

4. 任务背景：图片上传，选填，最多 1 张。
   - 4.1 推荐尺寸 750*352。
   - 4.2 待确认：图片格式、大小上限。

5. 任务说明：富文本编辑器，选填，用于配置用户侧任务说明。
   - 5.1 支持图文混排、基础文本样式、图片、视频、表格等富文本能力，以编辑器实际能力为准。

## 任务素材

6. 内容库：选择器，选填，点击后打开内容库选择弹窗。
   - 6.1 仅可选择启用状态的内容库。
   - 6.2 一次只能选择 1 个内容库。
   - 6.3 已选择内容库后，表单回显内容库名称。
   - 6.4 内容库禁用后，不再出现在选择弹窗中。

7. 默认素材：复选框，选填，默认不启用。
   - 7.1 勾选后展开默认素材配置字段。
   - 7.2 取消勾选后收起默认素材配置字段。
   - 7.3 启用时，当领取素材用户数大于内容库内容条数，系统统一分发默认素材内容。
   - 7.4 不启用时，当领取素材用户数大于内容库内容条数，系统不再分发默认素材。

8. 默认素材标题：文本输入，启用默认素材时必填，最多 200 字。
   - 8.1 支持中文、英文、数字、表情、符号。
   - 8.2 未启用默认素材时不展示、不校验。

9. 默认素材正文：多行文本，启用默认素材时必填，最多 800 字。
   - 9.1 支持换行、中文、英文、数字、表情、符号。
   - 9.2 未启用默认素材时不展示、不校验。

10. 默认素材标签：文本输入，选填。
    - 10.1 用于默认素材标签展示。
    - 10.2 未填写时不展示标签。

11. 默认素材封面图：图片上传，选填，最多 10 张。
    - 11.1 用于默认素材封面展示。
    - 11.2 待确认：图片格式、大小上限、尺寸建议。

12. 默认素材细节图：图片上传，选填，最多 10 张。
    - 12.1 用于默认素材细节展示。
    - 12.2 待确认：图片格式、大小上限、尺寸建议。

## 任务规则

13. 提交内容：配置项，条件必填，最多 5 个。
    - 13.1 与上传视图至少填写一个。
    - 13.2 点击添加入口后新增一条提交内容配置。

14. 上传视图：配置项，条件必填，最多 10 个。
    - 14.1 与提交内容至少填写一个。
    - 14.2 点击添加入口后新增一条上传视图配置。

## 任务奖励

15. 积分奖励：复选框，条件必填。
    - 15.1 与优惠券奖励至少勾选一种。
    - 15.2 勾选后可配置积分奖励信息。

16. 优惠券奖励：复选框，条件必填。
    - 16.1 与积分奖励至少勾选一种。
    - 16.2 勾选后可配置优惠券奖励信息。

# 列表交互

1. **选择内容库**：打开弹窗。
   - 1.1 点击「选择内容库」按钮，打开内容库选择弹窗。
   - 1.2 弹窗列表仅展示启用状态内容库。
   - 1.3 弹窗字段包含内容库、可分发、状态、选择。
   - 1.4 选择方式为单选。
   - 1.5 点击「确定」后回填所选内容库名称。
   - 1.6 点击「取消」后关闭弹窗，不变更已选内容库。

2. **启用默认素材**：展开/收起。
   - 2.1 勾选默认素材后，展开标题、正文、标签、封面图、细节图字段。
   - 2.2 取消勾选默认素材后，收起默认素材字段。

3. **保存**：点击「保存」按钮。
   - 3.1 触发表单校验，校验任务名称、活动时间、任务规则、任务奖励、默认素材字段。
   - 3.2 任务名称为空时，对应字段下方展示错误提示文案：「请输入任务名称」。
   - 3.3 活动时间为空时，对应字段下方展示错误提示文案：「请选择活动时间」。
   - 3.4 提交内容、上传视图均未填写时，展示错误提示：「提交内容、上传视图至少填写一个」。
   - 3.5 积分奖励、优惠券奖励均未勾选时，展示错误提示：「奖励方式至少勾选一种」。
   - 3.6 启用默认素材但默认素材标题为空时，展示错误提示：「请输入默认素材标题」。
   - 3.7 启用默认素材但默认素材正文为空时，展示错误提示：「请输入默认素材正文」。
   - 3.8 校验通过后调用保存接口，成功后 Toast 提示「保存成功」，返回积分任务列表。
   - 3.9 接口失败时 Toast 提示接口返回的错误信息。

4. **取消**：点击「取消」按钮。
   - 4.1 如表单无修改，直接返回积分任务列表。
   - 4.2 如表单有未保存内容，展示二次确认：「确认离开？当前修改将不会保存」。
   - 4.3 确认后返回积分任务列表，取消后留在当前页面。

5. **内容分发规则**：系统自动执行。
   - 5.1 用户参与积分任务领取素材时，系统优先从任务关联内容库中分发未领取内容。
   - 5.2 内容成功分发给用户后，该内容状态更新为已领取。
   - 5.3 系统记录领取用户、活动信息、领取时间。
   - 5.4 当内容库可分发内容不足且启用默认素材时，系统分发默认素材。
   - 5.5 当内容库可分发内容不足且未启用默认素材时，系统不分发素材。
