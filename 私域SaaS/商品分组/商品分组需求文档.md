# 商品分组需求文档

## 1. 业务背景

### 1.1 需求描述

商品系列、品牌、功效、使用场景较多，用户在小程序前端浏览商品时需要按分组快速筛选商品。后台需支持运营人员维护商品分组、子分组、前端展示样式、商品来源、权限范围及推广素材，并允许页面装修组件跳转到指定商品分组页。

### 1.3 目标结果

- 后台支持新建、编辑、启用、禁用、删除商品分组。
- 每个商品分组可维护多个子分组，用于前端二级筛选。
- 每个分组可配置搜索栏样式、商品展示样式、商品来源、商品信息展示项、权限范围。
- 小程序端可按后台配置展示商品分组页，用户可切换子分组并查看对应商品。
- 页面装修组件的跳转链接支持选择商品分组，并仅可选择启用中的分组页。
- 分组列表支持推广入口，运营可配置小程序分享卡片、海报、小程序路径和二维码相关内容。

### 1.4 本期范围

- 商品分组列表：查询、新增、编辑分组、配置商品、推广、启用/禁用、删除。
- 新增/编辑分组弹窗：维护分组名称和子分组，支持新增、单个删除、全部删除子分组。
- 配置商品页：搜索栏配置、分组商品配置、权限配置、左侧小程序预览。
- 分组商品配置：配置横向/左侧竖向筛选样式、双列/单列商品样式、商品名称/价格/加购按钮显示项。
- 按子分组配置商品来源：支持手动选择商品、商品分类。
- 商品选择弹窗：支持商品搜索、类型筛选、勾选商品、确认回填。
- 商品分类选择：支持后台分类多选，选择后收起下拉并展示已选分类。
- 小程序分组入口：后台菜单底部提供“小程序分组”，用于查看各分组的小程序展示效果。
- 推广弹窗：支持小程序链接与二维码、分享设置两类内容。
- 页面装修：页面装修列表、新增装修页；新增/编辑组件的跳转链接支持“商品分组”。

### 1.5 本期不做

- 不做批量导入分组、批量导入子分组。

## 2. 关键业务规则

### 4.1 唯一性 / 并发性

- 分组 ID 由系统生成，全局唯一，不允许人工编辑。
- 同一店铺内分组名称可重复；列表和页面装修选择项必须同时展示分组 ID 辅助识别。
- 子分组名称在同一分组内唯一不可重复。
- 同一分组下最多维护 20 个子分组。

### 4.2 字段定义

- 分组 ID：商品分组唯一标识。
  - 是否必填：系统生成。
  - 默认值：新建后自动生成。
  - 范围 / 长度：建议数字或字符串 ID，不超过 32 位。
  - 不允许：人工修改。

- 分组名称：前台页面标题和后台列表展示名称。
  - 是否必填：是。
  - 默认值：无。
  - 范围 / 长度：建议 1-20 个字符。
  - 不允许：纯空格、特殊不可见字符。

- 子分组：分组下的二级筛选项。
  - 是否必填：否。
  - 默认值：无。
  - 范围 / 长度：最多 20 个；单个名称建议 1-20 个字符。
  - 不允许：空值保存；超过上限继续新增。

- 状态：控制分组是否在前端可用。
  - 是否必填：是。
  - 默认值：新建后启用。
  - 可选值：启用、禁用。

- 创建人 / 创建时间：记录分组创建账号与时间。
  - 是否必填：系统记录。
  - 格式：账号名称；YYYY-MM-DD HH:mm:ss。

- 更新人 / 更新时间：记录最近一次修改账号与时间。
  - 是否必填：系统记录。
  - 触发：编辑分组、启用/禁用、配置商品、推广设置保存、删除前记录日志。

- 搜索栏显示配置：控制小程序分组页搜索栏是否显示。
  - 是否必填：是。
  - 默认值：显示。
  - 可选值：显示、隐藏。

- 搜索栏背景颜色：搜索区域背景色。
  - 是否必填：否。
  - 默认值：#ffffff。
  - 格式：HEX 色值。

- 搜索栏背景图：搜索栏区域背景图片。
  - 是否必填：否。
  - 格式：jpg/png。
  - 建议尺寸：宽 600px，高 800px。
  - 交互：上传后展示预览图，支持查看和删除。

- 搜索栏文字颜色：搜索栏占位文字颜色。
  - 是否必填：否。
  - 默认值：#666666。
  - 格式：HEX 色值。

- 搜索按钮图：搜索按钮自定义图片。
  - 是否必填：否。
  - 格式：jpg/png。
  - 建议尺寸：宽 166px，高 60px。
  - 交互：上传后展示预览图，支持查看和删除。

- 分组筛选样式：子分组在小程序端展示方式。
  - 是否必填：是。
  - 默认值：横向样式。
  - 可选值：横向样式、左侧竖向。

- 商品样式：商品列表布局方式。
  - 是否必填：是。
  - 默认值：双列。
  - 可选值：双列、单列展示。
  - 说明：单列滚动不作为本期配置项。

- 商品信息显示项：控制商品卡片信息。
  - 是否必填：否。
  - 默认值：商品名称、商品价格、加购按钮均显示。
  - 可选值：商品名称、商品价格、加购按钮。

- 子分组商品配置：每个子分组独立维护商品来源。
  - 是否必填：是。
  - 默认值：第一个子分组继承分组已有商品；其他子分组默认为空。
  - 说明：无子分组时使用“全部”作为默认配置项。

- 数据源类型：子分组商品来源。
  - 是否必填：是。
  - 默认值：手动选择。
  - 可选值：手动选择、商品分类。

- 商品列表：手动选择商品时维护。
  - 是否必填：否。
  - 范围 / 长度：商品数量无上限
  - 交互：选择后回填列表，支持上移、下移、删除。

- 选择分类：数据源类型为商品分类时维护。
  - 是否必填：否。
  - 可选范围：后台商品分类。
  - 交互：支持多选。

- 权限配置：控制分组前端可见用户。
  - 是否必填：是。
  - 默认值：全部用户。
  - 可选值：全部用户、对以下人群显示、对以下人群隐藏。
  - 选择用户群：当选择“对以下人群显示/隐藏”时启用。

- 推广分享标题：小程序分享卡片标题。
  - 是否必填：是。
  - 默认值：系统按分组名称生成默认文案。
  - 范围 / 长度：50 字以内。

- 推广分享图：小程序分享卡片图片。
  - 是否必填：否。
  - 建议比例：5:4。
  - 适用场景：分享微信好友、朋友圈。

- 推广海报：小程序分组海报。
  - 是否必填：否。
  - 建议尺寸：750px * 1080px。
  - 适用场景：分享海报、小程序码海报。

- 小程序路径：分组推广路径。
  - 是否必填：系统生成。
  - 示例：otherPackage/invite/index?id=100006。
  - 交互：支持复制。

- 页面装修跳转链接：装修组件点击后的跳转目标。
  - 是否必填：按组件配置要求。
  - 可选值：装修页、固定页面、品牌、后台分类、指定商品、我的小程序、其他小程序、h5链接、弹窗、视频号、数据引用、优惠券礼包、商品分组。
  - 商品分组规则：仅展示启用中的分组页，且只能选择 1 个。

### 4.3 计算规则

- 子分组信息展示：列表页最多直接展示前 5 个子分组名称，超过 5 个追加“.....”；鼠标悬停或点击可查看全部子分组。
- 商品排序：
  - 综合：按后台默认综合排序。
  - 销量：按销量从高到低。
  - 价格：前端展示价格排序入口，具体升降序切换规则待确认。
  - 新品：按商品上新时间倒序。
- 商品来源为手动选择时，商品展示顺序按运营配置顺序展示。
- 商品来源为商品分类时，按分类匹配商品集合展示，并叠加当前排序规则。
- 页面装修“商品分组”跳转下拉仅统计状态为启用的分组，禁用分组不展示。

### 4.4 关键联动逻辑

- 分组状态联动：
  - 启用：小程序分组页可展示，页面装修跳转链接可选择。
  - 禁用：小程序前端不可访问或不展示，页面装修跳转链接不可新选；已配置的装修链接保留，点击进页面商品为空。
  - 删除：列表移除，关联页面装修跳转目标商品为空。

- 子分组联动：
  - 新增/删除子分组后，配置商品页顶部子分组配置项同步变化。
  - 删除子分组后，该子分组下已配置商品关系删除。
  - 无子分组时，小程序预览不展示“全部”筛选项。
  - 有子分组时，横向样式展示“全部 + 子分组”；左侧竖向样式展示左侧列表。

- 搜索栏配置联动：
  - 修改背景颜色、背景图、文字颜色、搜索按钮图后，左侧预览实时刷新。
  - 背景图和搜索按钮图上传后支持查看、删除；删除后预览恢复默认样式。

- 商品样式联动：
  - 双列：商品卡片为双列白底卡片，上方主图、下方名称/价格/加购按钮。
  - 单列：商品卡片为横向布局，左侧主图、右侧商品信息和加购按钮。
  - 左侧竖向筛选 + 双列商品时，商品主图按正方形样式展示。

- 商品配置联动：
  - 切换子分组配置项时，仅展示当前子分组的数据源类型和商品配置。
  - 数据源类型为手动选择时，展示商品列表和“选择商品”入口。
  - 数据源类型为商品分类时，展示“选择分类”多选下拉。
  - 商品选择弹窗确认后，回填当前子分组商品列表。

- 权限配置联动：
  - 选择非“全部用户”时，用户群下拉为必填。

- 推广联动：
  - 推广弹窗包含“小程序链接与二维码”和“小程序分享设置”。
  - 分享设置中上传海报后，小程序链接与二维码页的海报预览同步更新。
  - 分享标题、分享图保存后，小程序分享卡片预览同步更新。

- 页面装修联动：
  - 组件配置中跳转链接选择“商品分组”后，出现分组页下拉框。
  - 分组页下拉仅展示启用中的分组，并限制单选 1 个。

### 4.5 异常处理规则

- 分组名称为空：保存失败，提示“请输入分组名称”。
- 子分组超过 20 个：禁止继续新增，提示“子分组最多添加20个”。
- 图片格式不支持：提示“仅支持 jpg/png 图片”。
- 图片上传失败：提示“上传失败，请重试”，保留原配置。
- 商品选择为空：允许保存；前端对应分组展示空态。
- 商品已下架/删除：前端不展示；后台商品列表直接移除。
- 商品分类为空：允许保存；前端展示空态。
- 页面装修跳转商品分组但未选择分组：发布失败，提示“请选择商品分组”。
- 页面装修已选择的商品分组被禁用或删除：发布/保存时提示分组不可用。

### 4.6 上限与极限（按需）

- 单个分组子分组最多 20 个。
- 分享标题最多 50 字。
- 搜索栏背景图建议 600px * 800px。
- 搜索按钮图建议 166px * 60px。
- 海报建议 750px * 1080px。
- 页面装修跳转商品分组仅支持选择 1 个。
- 手动选择商品数量、商品分类多选数量、分组总数无上限。

## 5. 状态与异常

### 5.1 业务状态

- 商品分组状态：启用、禁用。
- 商品来源状态：手动选择、商品分类。

### 5.2 页面默认态

- 商品分组列表默认展示全部分组，按创建/更新时间倒序展示。默认筛选项为空。
- 新增分组弹窗默认分组名称为空，子分组保留 1 个空输入行。
- 配置商品页默认进入“搜索栏”tab，左侧展示当前分组的小程序预览。
- 分组商品配置默认展示第一个子分组；无子分组时展示“全部”配置项。
- 小程序分组页面默认展示第一个可用分组的小程序效果。
- 推广弹窗默认进入“小程序链接与二维码”页。
- 页面装修列表默认展示全部页面。
- 新增装修页默认选中轮播组件，右侧展示轮播组件配置。

### 5.3 空态

- 商品分组列表无数据：表格展示“暂无数据”。
- 子分组为空：列表展示“未配置”；小程序端不展示子分组筛选栏。
- 当前子分组无商品：小程序商品区展示空态，文案建议“暂无商品”。
- 商品选择弹窗无匹配商品：展示“暂无商品”。
- 商品分类树无数据：展示“暂无分类”。
- 小程序分组无分组：展示“暂无商品分组”。
- 页面装修列表无数据：展示“暂无数据”。

### 5.4 异常态 / 失败态

- 保存失败：保留当前填写内容，提示失败原因。


### 5.6 状态-操作矩阵（按需）

| 状态 | 可执行操作 | 备注 |
| --- | --- | --- |
| 启用 | 编辑分组、配置商品、推广、禁用、删除、页面装修选择 | 前端可展示 |
| 禁用 | 编辑分组、配置商品、推广、启用、删除 | 数据保留，前端不展示；页面装修不可新选择 |
| 已删除 | 无 | 从列表移除；关联关系需按异常规则处理 |

### 5.7 状态流转（按需）

- 商品分组新建 → 启用：新建保存成功后默认启用。
- 启用 → 禁用：列表开关关闭，分组数据保留，前端不展示。
- 禁用 → 启用：列表开关开启，前端恢复展示。
- 启用/禁用 → 删除：二次确认后移除列表数据。

## 6. 衍生需求与默认授权项

### 6.1 导出

- 本期商品分组列表未要求导出。
- 若后续增加导出，建议导出字段包含：分组 ID、分组名称、子分组信息、状态、创建人、创建时间、更新人、更新时间。
- 建议文件名：商品分组_{时间戳}.xlsx。

### 6.2 日志

- 商品分组新建、编辑、删除、启用、禁用需记录操作日志。
- 日志字段建议：日志 ID、操作人、操作描述、提示信息、操作时间。

### 6.3 ID 规则

- 商品分组 ID：系统生成，建议使用自增 ID 或业务前缀 + 序列号。
- 子分组可使用系统内部 ID，前端展示名称。
- 小程序路径中的 id 参数应使用可定位分组或推广页的业务 ID。
