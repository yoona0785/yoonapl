# 用户运营需求文档

## 1. 业务背景

系统面向指定用户、指定用户群进行运营触达，支持发放优惠券、发放积分、发送小程序订阅消息、发送短信通知。运营计划支持立即执行、定时执行、周期执行，周期执行可按每周、每月、每几天配置执行时间。

### 1.1 需求描述

1. 用户中心新增用户运营菜单。
2. 用户运营支持计划列表、搜索、新建、编辑、复制、删除、日志、立即执行、重新执行。
3. 用户运营支持订阅模板、短信模板管理，模板列表以独立列表页展示，新增/编辑模板使用弹窗配置。
4. 运营计划支持指定手机号用户、选择用户群。
5. 发放权益支持优惠券、积分，可单独配置或组合配置。
6. 通知方式支持小程序订阅消息、短信通知，可单独配置或组合配置。
7. 运营计划支持立即执行、定时执行、周期执行。
8. 周期执行支持配置执行时间范围，范围内可配置每周几的几点、每月几号的几点、每几天的几点。
9. 用户运营列表展示预估/实际执行人群，点击后跳转至用户中心-用户管理，并按该计划执行人群过滤用户列表。

### 1.2 目标结果

1. 实现用户权益发放。
2. 实现用户消息通知。
3. 支持运营计划执行前预估触达人群、执行后查看实际执行人群。

### 1.3 本期范围

1. 用户运营计划列表、搜索、新建、编辑、复制、删除、日志。
2. 计划执行方式：立即执行、定时执行、周期执行。
3. 计划人群：指定手机号用户、指定用户群。
4. 发放权益：优惠券、积分。
5. 通知渠道：小程序订阅消息、短信通知。
6. 订阅模板列表、短信模板列表，以及对应新增/编辑弹窗。
7. 点击预估/实际执行人群跳转用户管理列表并默认过滤。

### 1.4 本期不做

1. 不做发放权益的回收。

## 2. 页面路径与页面结构

### 2.1 用户运营列表

页面路径：用户中心 / 用户运营

列表字段：

| 字段 | 说明 |
| --- | --- |
| 计划名称 | 展示计划名称、计划ID、执行状态标签 |
| 计划类型 | 指定手机号用户 / 指定用户群 |
| 计划方式 | 立即执行 / 定时执行 / 周期执行 |
| 计划时间 | 展示立即执行时间、定时执行时间或周期规则 |
| 发放权益 | 展示优惠券、积分配置 |
| 通知方式 | 展示小程序订阅消息、短信通知 |
| 预估/实际执行人群 | 待执行展示预估人数；执行成功/执行失败展示实际人数 |
| 状态 | 待执行 / 执行失败 / 执行成功 |
| 操作 | 按状态展示可操作项 |

搜索条件：

1. 计划名称。
2. 状态：待执行 / 执行失败 / 执行成功。
3. 更多搜索：计划类型、计划方式。

说明：

1. 点击预估/实际执行人群，进入用户中心-用户管理列表。
2. 用户管理列表默认带入该计划的人群手机号或用户群过滤条件。

### 2.2 用户管理列表联动

页面路径：用户中心 / 用户管理

从用户运营计划进入用户管理时：

1. 指定手机号用户：默认带入手机号过滤。
2. 指定用户群：默认按用户群过滤。

### 2.3 新建/编辑运营计划

页面路径：用户中心 / 用户运营 / 新增计划

表单分区：

1. 基本信息。
2. 计划人群。
3. 权益礼包。
4. 通知配置。
5. 计划时间。

字段规则：

| 字段 | 规则 |
| --- | --- |
| 计划名称 | 必填，最多50个字，不展示额外说明文案 |
| 计划方式 | 必填，立即执行 / 定时执行 / 周期执行 |
| 计划类型 | 必填，指定手机号用户 / 指定用户群 |
| 手机号用户 | 选择指定手机号用户时展示，支持录入手机号 |
| 用户群 | 选择指定用户群时展示，支持选择已有用户群 |
| 优惠券配置 | 勾选后展示包含优惠券表格 |
| 积分配置 | 勾选后填写发放积分数 |
| 通知渠道 | 小程序订阅消息 / 短信通知 |
| 计划时间 | 根据计划方式展示对应时间配置 |

## 3. 权益礼包规则

### 3.1 优惠券配置

1. 勾选发放优惠券后，展示包含优惠券区域。
2. 包含优惠券区域默认表格空态。
3. 点击添加优惠券，打开选择优惠券弹窗。
4. 选择优惠券弹窗支持设置赠券数。
5. 赠券数不可小于1，不可超过优惠券剩余发放数量。
6. 确认后回填至包含优惠券表格。
7. 回填后可直接编辑赠券数。
8. 回填后可删除已选优惠券。
9. 勾选发放优惠券但未添加优惠券时，不允许保存。

优惠券回填表格字段：

| 字段 | 说明 |
| --- | --- |
| 赠券数 | 可编辑 |
| 优惠券名称 | 展示券类型、券名称、券ID |
| 优惠信息 | 展示优惠内容 |
| 优惠承担 | 展示平台/供应商承担比例 |
| 券有效期 | 展示固定有效期或领取后有效期 |
| 发券量/领取数 | 展示发券量与领取数 |
| 操作 | 删除 |

### 3.2 积分配置

1. 勾选发放积分后，展示积分数量输入框。
2. 积分数量必须大于0。
3. 未勾选发放积分时，不保存积分配置。

---

## 4. 通知模板规则

### 4.1 模板列表

1. 点击用户运营列表右上角订阅模板，进入订阅模板列表页。
2. 点击用户运营列表右上角短信模板，进入短信模板列表页。
3. 模板列表页支持搜索、状态筛选、新增、编辑、删除。
4. 新增/编辑模板仍使用弹窗配置。

### 4.2 订阅模板配置

字段规则：

| 字段 | 规则 |
| --- | --- |
| 模板ID | 下拉单选，可选所有已添加的模板ID |
| 模板字段1 | 选择模板ID后带出可配置字段 |
| 模板字段2 | 选择模板ID后带出可配置字段 |
| 详情 | 只可填写小程序路径 |
| 状态 | 默认开启，可选开启、禁用 |

校验规则：

1. 模板ID必填。
2. 详情必填。
3. 详情必须为小程序路径，路径以 /pages/ 开头。

### 4.3 短信模板配置

字段规则：

| 字段 | 规则 |
| --- | --- |
| 模板CODE | 下拉单选，可选所有已添加的模板CODE |
| 模板内容 | 500字内，选择模板CODE后带出可配置内容 |
| 状态 | 默认开启，可选开启、禁用 |

短信条数规则：

1. 模板内容最多500字。
2. 120个字算1条短信。
3. 输入模板内容时实时展示当前字数和预计短信条数。

---

## 5. 执行方式规则

### 5.1 立即执行

1. 保存后立即进入执行队列。
2. 执行完成后状态为执行成功或执行失败。
3. 列表展示实际执行人群。

### 5.2 定时执行

1. 需配置执行时间。
2. 执行前状态为待执行。
3. 到达执行时间后进入执行队列。
4. 执行完成后状态为执行成功或执行失败。

### 5.3 周期执行

1. 需配置执行时间范围。
2. 周期规则支持每周、每月、每几天。
3. 周期内按配置时间执行。
4. 执行前状态为待执行。
5. 执行完成后状态为执行成功或执行失败。

---

## 6. 状态与操作

### 6.1 业务状态

| 状态 | 说明 |
| --- | --- |
| 待执行 | 定时或周期计划尚未执行，或立即计划尚未进入执行完成态 |
| 执行成功 | 计划已完成权益发放或消息通知 |
| 执行失败 | 计划执行失败，可重新执行 |

### 6.2 状态-操作矩阵

| 状态 | 可执行操作 | 备注 |
| --- | --- | --- |
| 待执行 | 复制、编辑、立即结束、立即执行、日志、删除 | 可提前执行或结束 |
| 执行成功 | 复制、日志、删除 | 不可编辑，不可重新执行 |
| 执行失败 | 复制、重新执行、日志、删除 | 重新执行后进入执行成功或执行失败 |

### 6.3 状态流转

1. 新建立即执行计划：保存后进入执行队列，完成后变为执行成功或执行失败。
2. 新建定时执行计划：保存后为待执行，到达执行时间后进入执行队列。
3. 新建周期执行计划：保存后为待执行，到达周期执行时间后进入执行队列。
4. 待执行点击立即执行：进入执行队列，完成后变为执行成功或执行失败。
5. 待执行点击立即结束：计划结束，不再等待执行。
6. 执行失败点击重新执行：重新进入执行队列。
7. 删除：删除计划记录，不做权益回收。

---

## 7. 异常与校验

1. 计划名称为空时，不允许保存。
2. 未配置权益且未配置通知时，不允许保存。
3. 勾选发放优惠券但未添加优惠券时，不允许保存。
4. 勾选发放积分但未填写积分数量时，不允许保存。
5. 勾选通知渠道但未选择对应模板时，不允许保存。
6. 周期执行结束时间不得早于开始时间。
7. 已执行成功计划不可编辑。
8. 删除、立即结束需二次确认。

---

## 8. 日志与数据

### 8.1 操作日志

以下操作需记录日志：

1. 新建计划。
2. 编辑计划。
3. 删除计划。
4. 立即结束。
5. 立即执行。
6. 重新执行。
7. 新增/编辑/删除订阅模板。
8. 新增/编辑/删除短信模板。

### 8.2 数据来源

1. 用户群来源于用户中心-用户群。
2. 手机号用户由运营人员录入。
3. 优惠券来源于营销中心-优惠券。
4. 订阅模板来源于已添加的小程序订阅模板。
5. 短信模板来源于已添加的短信模板CODE。
6. 执行人群数据联动用户中心-用户管理。
