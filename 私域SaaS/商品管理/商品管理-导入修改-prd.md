# 商品管理-批量修改功能需求

## 1. 需求背景

商品初始化创建后，缺少后台 SPU、属性名称、品牌、后台分类、发货仓、运费模板、规格数量、成本价、售价、重量、限购等信息。运营如果逐个进入商品编辑页处理，操作成本高，且容易出现漏改、错改。

因此需要在商品管理列表增加「导入修改」能力，通过文件导入方式按商品 ID 批量更新不同商品的不同字段。

## 2.目标结果

- 支持运营通过导入文件批量修改商品信息。
- 支持不同商品配置不同 SPU、品牌、分类、规格和价格。
- 支持导入前校验、失败数据前置展示、失败列表下载。
- 导入表中字段未填写时视为不更新，避免覆盖原商品已有数据。

## 3.本期范围

- 商品管理列表增加「导入修改」入口。
- 导入修改弹窗支持选择导入内容：基础信息、价格信息。
- 支持下载导入模板、选择导入文件、点击校验、查看导入预览、下载失败列表、确认修改。
- 基础信息支持按商品 ID 修改：后台 SPU、属性名称、品牌、后台分类、发货仓、运费模板。
- 价格信息支持按商品 ID 修改：规格数量、成本价、售价、重量、限购。
- 价格信息支持同一商品 ID 多行导入，一行代表一个规格数量。

## 4.本期不做

- 不做导入历史记录页。商品操作日志记录到每个商品上

## 2. 目标端

PC 管理后台。

### 2.2 使用角色

- 商家：在商品管理列表发起导入修改，上传文件并确认修改。

## 3. 关键业务规则

###  唯一性 / 并发性

- 导入修改按商品 ID 匹配商品。
- 同一商品 ID 在价格信息导入中允许出现多行。

### 字段定义

- 导入内容：单选项，必选。
  - 可选值：基础信息、价格信息。
  - 默认值：基础信息。
  - 选择基础信息时，导入表必须使用基础信息模板。
  - 选择价格信息时，导入表必须使用价格信息模板。

- 基础信息模板字段：
  - 商品ID：必填，系统内必须存在。
  - 后台SPU：选填，系统内必须存在。
  - 属性名称：选填，系统内必须存在。
  - 品牌：选填，系统内必须存在。
  - 后台分类：选填，系统内必须存在。
  - 发货仓：选填，系统内必须存在。
  - 运费模板：选填，系统内必须存在。

- 价格信息模板字段：
  - 商品ID：必填，系统内必须存在。
  - 规格数量：必填，只能填写正整数。
  - 成本价：选填，填写时必须为正数，最多 2 位小数。
  - 售价：选填，填写时必须为正数，最多 2 位小数。
  - 重量：选填，填写时必须为正数，最多 2 位小数。
  - 限购：选填，填写时只能填写正整数。



### 4.3 计算规则

- 价格信息规格更新：
  - 导入后的规格数量集合作为该商品更新后的规格集合。
  - 原商品规格数量有 1、2、3、4，导入为 1、2、4 时，规格 3 删除。
  - 原商品规格数量有 1、2，导入为 1、2、3、4 时，规格 3、4 新增。
  - 导入表中某字段未填写时，该字段不更新。

### 4.4 关键联动逻辑

- 导入内容与模板联动：
  - 选择基础信息后，下载模板为基础信息模板。
  - 选择价格信息后，下载模板为价格信息模板。
  - 上传文件表头与导入内容不一致时，点击校验提示「请选择正确的导入修改表」。

- 基础信息字段互斥：
  - 填写后台 SPU 或属性名称时，不可填写品牌、后台分类。
  - 填写品牌或后台分类时，不可填写后台 SPU、属性名称。
  - 填写属性名称时，必须填写后台 SPU。
  - 属性名称必须属于所填写后台 SPU 关联的属性名称。

- 导入预览展示：
  - 未上传文件时，导入预览提示「请先上传导入文件，然后点击校验。」。
  - 点击校验后展示导入预览表格。
  - 校验失败的数据展示在前，校验通过的数据展示在后。
  - 存在失败数据时，在导入预览标题右侧展示「下载失败列表」。

- 确认修改：
  - 未点击校验时，不允许确认修改。
  - 存在失败数据时，不允许确认修改。
  - 全部校验通过后，点击「确认修改」弹出二次确认。
  - 二次确认通过后执行数据更新。

### 4.5 异常处理规则

- 未上传文件点击校验：提示请先上传导入文件。
- 导入表与导入内容不一致：Toast 提示「请选择正确的导入修改表」。
- 商品 ID 为空：该行校验失败。
- 商品 ID 不存在：该行校验失败，失败原因展示「商品不存在」。
- 基础信息字段互斥冲突：该行校验失败，失败原因展示字段冲突原因。
- 属性名称与后台 SPU 不匹配：该行校验失败，失败原因展示属性不属于该 SPU。
- 规格数量为空或非正整数：该行校验失败，失败原因展示「规格数量必填」或「规格数量只能填写正整数」。
- 成本价、售价、重量格式错误：该行校验失败，失败原因展示「需填写正数且最多 2 位小数」。
- 限购格式错误：该行校验失败，失败原因展示「限购只能填写正整数」。
- 存在失败数据时点击确认修改：提示存在失败数据时不可确认修改。

## 5. 导出 / 下载
- 下载模板：按当前选择的导入内容下载对应 CSV 模板。
- 下载失败列表：仅在存在失败数据时展示，下载内容包含原导入字段与失败原因。
- 文件名可默认：
  - 基础信息模板：`商品导入修改-基础信息.csv`
  - 价格信息模板：`商品导入修改-价格信息.csv`
  - 失败列表：`商品导入修改-失败列表.csv`
