
## 1. 业务背景


### 1.1 需求描述

现有运费模板已支持商品批次运费、统一运费、按重量计费、按件计费。本期新增“按金额计费”方式，支持按配送区域配置金额门槛、未达门槛运费和达到门槛后的运费。

### 1.3 目标结果

1. 按金额计费均支持按省/市/区三级配送区域配置差异化运费。

### 1.4 本期范围

- 运费模板新增/编辑：支持商品批次运费、统一运费、按重量计费、按件计费、按金额计费五种计费方式。
- 运费模板新增/编辑：按重量计费、按件计费、按金额计费支持配置可配送区域和运费规则。

## 2. 关键业务规则

### 4.1 字段定义

- 模板名称：运费模板名称。
  - 是否必填：是
  - 默认值：无
  - 范围 / 长度：50 字以内
  - 不允许：不允许为空，不允许与已有模板名称重复

- 计费方式：运费计算方式。
  - 是否必填：是
  - 默认值：无
  - 范围 / 长度：商品批次运费 / 统一运费 / 按重量计费 / 按件计费 / 按金额计费
  - 不允许：不允许为空

- 快递公司：模板关联快递公司。
  - 是否必填：是
  - 默认值：无
  - 范围 / 长度：普通快递、EMS邮政速递、YTO圆通速递、顺丰、中通、STO申通快递、韵达等写死的快递公司
  - 不允许：不允许为空

- 金额门槛（元）：按金额计费的订单金额判断门槛。
  - 是否必填：按金额计费时必填
  - 默认值：1.00
  - 范围 / 长度：金额，单位元，支持两位小数，大于等于 0
  - 不允许：不允许为空，不允许小于 0

- 不满金额运费（元）：订单金额未达到金额门槛时收取的运费。
  - 是否必填：按金额计费时必填
  - 默认值：0.00
  - 范围 / 长度：金额，单位元，支持两位小数，大于等于 0
  - 不允许：不允许为空，不允许小于 0

- 满金额运费（元）：订单金额达到金额门槛时收取的运费。
  - 是否必填：按金额计费时必填
  - 默认值：0.00
  - 范围 / 长度：金额，单位元，支持两位小数，大于等于 0
  - 不允许：不允许为空，不允许小于 0

### 4.2 计算规则

- 按金额计费：
  - 规则：按可配送区域配置金额门槛、不满金额运费、满金额运费。
  - 示例：金额门槛 99 元，不满金额运费 8 元，满金额运费 0 元；订单金额 88 元收取 8 元运费，订单金额 99 元收取 0 元运费。
  - 边界说明：订单金额等于金额门槛时，按满金额运费计算。

### 4.3 异常处理规则

- 运费规则未选择可配送区域时，禁止保存并提示“请选择可配送区域”。
- 金额、重量、件数类字段为空或小于 0 时，禁止保存并提示对应字段填写错误。

### 4.4 上限与极限（按需）

- 模板名称最多 50 字。
- 快递公司至少选择 1 个，上限以后端配置为准。
- 运费规则至少配置 1 条；单模板内规则数量上限待后端接口确认。
- 金额字段单位为元，最多两位小数。
