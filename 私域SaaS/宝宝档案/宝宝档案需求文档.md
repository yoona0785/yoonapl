# 宝宝档案需求文档

更新日期：2026-06-11

## 1. 业务背景

### 1.1 需求描述

平台需要收集、维护用户的宝宝档案信息。宝宝信息由用户在小程序端填写，也支持运营人员在 PC 后台用户详情中查看和编辑。

本期将宝宝信息填写方式调整为可配置模式：平台可在系统配置中开启或关闭宝宝信息填写，并选择使用固定宝宝信息页或页面装修中的表单组件页。页面装修表单组件支持配置字段、字段顺序、字段展示逻辑、背景样式、提交跳转和可见权限。

宝宝档案字段沉淀到用户信息后，可在用户中心的标签、用户群中作为指标条件使用，用于自动打标和圈选用户。

### 1.2 目标结果

1. 平台可配置是否开启宝宝信息填写。
2. 平台可选择宝宝信息填写页类型：固定宝宝信息页、装修页。
3. 页面装修支持新增表单组件，并可配置宝宝档案字段。
4. 小程序端根据后台配置展示宝宝档案填写入口、表单页和已填写信息。
5. PC 后台用户详情可查看、编辑用户填写的宝宝信息。
6. 标签和用户群可使用宝宝档案字段作为指标条件。
7. 系统配置恢复内容配置页面，并移除 009 添加敏宝跳转页配置项。

### 1.3 本期范围

1. 系统配置-业务配置新增宝宝信息配置。
2. 商城装修-页面装修新增列表页与表单装修能力。
3. 小程序端新增宝宝档案填写入口、表单填写、表单修改和已填写展示。
4. 用户中心-用户管理新增列表页和用户详情宝宝信息展示、编辑。
5. 用户中心-标签新增列表页、新建/编辑页，并支持宝宝档案及全部指标配置。
6. 用户中心-用户群新增列表页、新建/编辑页，并支持宝宝档案及全部指标配置。
7. 系统配置-内容配置更新配置页列表。

### 1.4 本期不做

1. 不做用户群运营发积分、发优惠券等营销动作。
2. 不做宝宝信息变更的消息通知、短信通知、企微通知。
3. 不做宝宝档案字段的批量导入。

## 3. 系统配置-业务配置

### 3.1 宝宝信息配置位置

宝宝信息配置展示在系统配置-业务配置页面的客服配置下方、平台信息上方。

### 3.2 配置字段

| 字段 | 类型 | 是否必填 | 展示规则 | 说明 |
| --- | --- | --- | --- | --- |
| 信息填写 | 单选 | 是 | 固定展示 | 可选开启、关闭 |
| 填写页 | 下拉选择 | 开启后必填 | 信息填写为开启时展示 | 可选固定宝宝信息页、装修页 |
| 填写跳转 | 勾选 | 否 | 填写页为固定宝宝信息页时展示 | 可勾选开启敏宝跳转、未开启敏宝跳转 |
| 开启敏宝跳转 | 跳转配置 | 勾选后必填 | 勾选开启敏宝跳转时展示 | 按选择的跳转类型及参数跳转 |
| 未开启敏宝跳转 | 跳转配置 | 勾选后必填 | 勾选未开启敏宝跳转时展示 | 配置内容同开启敏宝跳转 |

### 3.3 字段说明

1. 信息填写开启后，填写页后展示提示：`注：固定信息包含昵称、性别、生日，通过页面装修-表单组件 可自定义宝宝信息字段`。
2. 信息填写关闭时，小程序端不展示宝宝档案填写入口。
3. 填写页选择装修页时，小程序端读取页面装修的配置。
4. 填写页选择固定宝宝信息页时，小程序端展示固定宝宝信息页，并可根据填写跳转配置跳转。

### 3.4 填写跳转配置

1. 字段名称展示为`填写跳转`。
2. 字段类型展示为勾选项，可选`开启敏宝跳转`、`未开启敏宝跳转`，均非必选。
3. 勾选任一项后，在下方展示对应配置区。
4. 开启敏宝跳转配置右侧展示提示：`注：开启敏宝跳转后，按所选跳转类型及参数跳转`。
5. 未开启敏宝跳转配置右侧展示提示：`注：未开启敏宝跳转后，按所选跳转类型及参数跳转`。

## 4. 商城装修-页面装修

### 4.1 表单组件

页面路径：商城装修-页面装修-新增/编辑装修页。

#### 4.1.1 组件入口

1. 左侧组件库新增`表单`组件。
2. 选择表单组件后，中间展示小程序页面预览，右侧展示表单配置。

#### 4.1.2 表单配置顺序

表单配置按以下顺序展示：

1. 组件备注
2. 背景颜色
3. 背景图
4. 表单字段
5. 跳转类型
6. 权限管理

#### 4.1.3 基础配置字段

| 字段 | 类型 | 是否必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| 组件备注 | 输入框 | 否 | 空 | 50 字内，用于后台识别组件 |
| 背景颜色 | 颜色选择 | 否 | #FFFFFF | 填充到字段卡片外 |
| 背景图 | 图片上传 | 否 | 空 | 上传后覆盖背景颜色，并填充到字段卡片外 |


### 4.2 表单字段

#### 4.2.1 添加字段

1. 点击添加字段，弹窗展示可添加字段。
2. 字段按一级分类分组。
3. 一级分类按分类名前的数字编码排序，编码越小越靠前。
4. 分组名展示时去掉`01-`、`02-`等数字编码。
5. 选择字段后返回编辑区生成字段卡片。

#### 4.2.2 字段卡片

每个字段以独立卡片形式展示，字段卡片包含：

| 内容 | 说明 |
| --- | --- |
| 字段名称 | 展示内置字段名称和字段类型 |
| 拖拽点 | 支持拖动调整字段顺序 |
| 必填开关 | 控制字段必填 / 非必填 |
| 删除 | 点击后无需确认，直接删除字段 |
| 更多 | 打开字段显示逻辑弹窗 |
| 前台展示名称 | 标准输入框，默认取字段名称 |
| 填写提示 | 标准输入框，默认空 |

#### 4.2.3 必填开关

1. 开启表示字段必填，展示蓝底，文案`必填`。
2. 关闭表示字段非必填，展示浅灰底，文案`非必填`。

#### 4.2.4 字段排序

1. 字段卡片支持拖动排序。
2. 小程序表单字段展示顺序与后台表单配置顺序一致。
3. PC 用户详情编辑弹窗字段顺序与字段库顺序一致。

### 4.3 字段显示逻辑

#### 4.3.1 弹窗名称

点击字段卡片更多按钮，打开弹窗，弹窗名称为`字段显示逻辑`。

#### 4.3.2 可选字段

字段显示逻辑中可选字段范围为宝宝档案字段类型为以下类型的字段：

1. 数字
2. 日期
3. 单选
4. 多选
5. 级联选择

#### 4.3.3 条件配置

1. 支持设置`满足以下所有条件`或`满足以下任一条件`。
2. 条件行包含：字段、条件、字段填写值。
3. 单选、多选字段的填写值取字段选项。
4. 数字字段的填写值为数字输入框。
5. 日期字段的填写值为日期选择框。
6. 条件包括等于、包含、大于等于、小于等于等，实际可选项按字段类型展示。

### 4.4 内置字段

字段类型包含：文本、数字、日期、时间、单选、多选、级联选择、图片上传、视频上传、评分、NPS 净推荐值等。

#### 4.4.1 基础信息字段排序

基础信息字段按以下顺序展示：

1. 宝宝排行，单选，可选大宝、二宝、三宝
2. 宝宝昵称，文本
3. 宝宝性别，单选，可选男宝宝、女宝宝
4. 出生日月，日期，到年月日
5. 头像，图片上传
6. 用户姓名，文本
7. 用户手机号，数字
8. 当前阶段，单选，可选宝宝已出生、孕期、备孕
9. 预产期，日期，到年月日
10. 宝宝关系，单选，可选妈妈、爸爸、宝家人、其他
11. 所在城市，级联选择，可选省市区

## 5. 小程序宝宝档案

### 5.1 入口规则

1. 当系统配置-宝宝信息-信息填写为关闭时，小程序不展示宝宝档案填写入口。
2. 当信息填写为开启，且填写页为装修页时，小程序我的-个人信息页展示宝宝档案入口。
3. 如平台存在多个可填写表单入口，小程序-我的信息页只展示宝宝信息关联的页面填写入口及填写后的信息查看入口；其他表单填写的宝宝信息若已填写则展示信息，未填写不展示入口
4. 点击入口进入对应表单填写页。
5. 填写完成后返回我的-个人信息页展示已填写宝宝卡片。
6. 已填写卡片点击后进入表单填写页，可修改并重新提交。

### 5.2 未填写入口样式

| 元素 | 规格 |
| --- | --- |
| 标题 | 宝宝档案 |
| 主文案 | 点击填写档案|
| 默认头像 | 按UI设计尺寸|

### 5.3 表单填写页

1. 字段按后台表单配置顺序展示。
2. 字段名称按后台配置的前台展示名称展示。
3. 填写提示按后台配置展示，默认空。
4. 字段类型需真实交互，至少支持文本、数字、日期、单选、多选、级联选择、图片上传。
5. 必填字段未填写时不可提交，并提示用户填写。
6. 配置字段显示逻辑后，小程序端需按用户输入实时控制字段展示。
7. 背景颜色或背景图按后台表单组件配置渲染。

### 5.4 已填写宝宝卡片

填写宝宝信息后，在我的-个人信息页统一展示宝宝卡片。

#### 5.4.1 已填写入口样式

| 元素 | 规格 |
| --- | --- |
| 标题 | 宝宝档案 |
| 宝宝姓名 | 填写的昵称 |
| 年龄与性别 | 根据生日计算 |
| 性别图标 | 男宝宝蓝色 #4A90E2，女宝宝粉色 |
| 头像 | 按UI设计尺寸|

#### 5.4.1 兜底规则

1. 未填写昵称时展示`暂无`。
2. 未填写生日或无法计算月龄时，月龄展示`暂无`。
3. 未填写性别时，不展示性别文案和性别图标。
4. 未上传头像时展示默认头像。

### 5.5 多条记录信息

1. 用户可存在多条宝宝信息记录。
2. 我的-个人信息页按记录展示多个宝宝卡片。
3. 用户详情页宝宝信息列表同步展示多条记录。
4. 每条记录点击后进入对应表单修改。

## 6. 用户中心-用户管理

### 6.1 用户管理列表

页面路径：用户中心-用户管理。

### 6.2 用户详情-宝宝信息

页面路径：用户中心-用户管理-用户详情。

#### 6.2.1 宝宝信息列表字段

| 字段 | 取值规则 |
| --- | --- |
| 宝宝姓名 | 取宝宝昵称，未填写展示 `-` |
| 生日 | 取出生日月，未填写展示 `-` |
| 月龄 | 按生日计算，未填写生日展示 `-` |
| 性别 | 取宝宝性别，未填写展示 `-` |
| 牛奶蛋白过敏 | 取是否确诊牛奶蛋白过敏，未填写展示 `-` |
| 特配奶粉 | 取当前奶粉阶段，未填写展示 `-` |
| 操作 | 编辑 |

### 6.3 编辑宝宝信息弹窗

1. 点击宝宝信息列表中的编辑打开弹窗。
2. 弹窗标题为`编辑宝宝信息`。
3. 字段和顺序与表单配置中可添加的全部宝宝档案字段一致。
4. 支持编辑宝宝所有信息字段。
5. 控件类型按字段类型展示。
6. 保存成功后更新用户详情宝宝信息列表。
7. 未填写字段在列表中展示为`-`。

## 7. 用户中心-标签

### 7.1 标签列表

页面路径：用户中心-标签。

### 7.2 新建 / 编辑标签

1. 点击新增标签进入新建标签页。
2. 点击编辑进入编辑标签页。
3. 标签基础信息包含标签名称、标签说明、标签类型。
4. 标签类型选择自动打标时，展示满足条件配置区。
5. 满足条件支持所有条件全部满足、任一条件满足。
6. 右侧指标选择支持搜索指标名称。

### 7.3 指标选择规则

1. 指标大类取指标表的一级分类。
2. 一级分类按分类名前的数字编码排序，编码越小越靠前。
3. 大类展示时去掉`01-`、`02-`等数字编码。
4. 大类下的指标分类取指标表的二级分类。
5. 二级分类按分类名前的数字编码排序，编码越小越靠前。
6. 二级分类展示时去掉数字编码。
7. 宝宝档案作为一个指标大类展示。
8. 宝宝档案可选附件表格中的宝宝信息字段作为指标。
9. 指标名称取表格中的指标名称。
10. 指标分类取二级分类名称。

### 7.4 指标定义规则

1. 勾选指标后，在左侧生成指标定义卡片。
2. 条件下拉取表格`条件定义`字段。
3. 条件定义为空时，表示该指标无条件下拉。
4. 指标值取表格`指标值`字段。
5. 指标值为单选时，只能选择 1 个值。
6. 指标值为多选时，可选择多个值。
7. 非区间条件只有 1 个输入框。
8. 区间条件有 2 个输入框，分别填写起始值和结束值。
9. 时间 + 次数 / 金额类指标，需展示时间条件和次数 / 金额条件。
10. 时间 + 商品 + 次数 / 金额类指标，需展示时间条件、商品条件和次数 / 金额条件。

## 10. 数据与计算规则

### 10.1 数据保存

1. 小程序提交的宝宝信息需记录到用户信息。
2. 每条宝宝信息记录需保存来源表单页、字段 ID、字段名称、字段类型、字段值、提交时间、更新时间。
3. PC 后台编辑宝宝信息后需更新同一条宝宝信息记录。
4. 表单字段删除后，不影响历史已填写数据；前台展示按当前有效配置展示。

### 10.2 月龄计算

1. 月龄依据出生日月和当前日期计算。
2. 出生日月未填写时，月龄展示`-`或`暂无`，按所在页面兜底规则展示。
3. 小程序宝宝卡片中无月龄时展示`暂无`。

### 10.3 缺省值

1. PC 后台列表字段无值时展示`-`。
2. 小程序宝宝卡片无昵称时展示`暂无`。
3. 小程序宝宝卡片无性别时不展示性别。
4. 小程序宝宝卡片无头像时展示默认头像。

### 10.4 字段显示逻辑

1. 字段显示逻辑在小程序填写页实时生效。
2. 被隐藏字段不进行必填校验。
3. 被隐藏字段已填写的值不在本次提交中作为展示依据，实际是否清空由接口设计确认。

## 11. 异常与校验

### 11.1 业务配置校验

1. 信息填写开启时，填写页不能为空。
2. 勾选开启敏宝跳转时，开启敏宝跳转配置必填项不能为空，否则提示`请完善跳转页配置`。
3. 勾选未开启敏宝跳转时，未开启敏宝跳转配置必填项不能为空，否则提示`请完善未开启敏宝跳转配置`。

### 11.2 表单配置校验

1. 组件备注最多 50 字。
2. 背景颜色需为合法颜色值。
3. 背景图上传失败时保留原配置并提示失败。
4. 前台展示名称不能为空。
5. 必填字段在小程序端未填写时不可提交。
6. 字段显示逻辑条件不完整时不可保存。

### 11.3 指标配置校验

1. 区间条件需填写两个值。
2. 区间最小值不能大于最大值。
3. 时间区间开始时间不能晚于结束时间。
4. 多选指标至少选择 1 个值。
5. 已选择指标删除时同步移除左侧定义条件。
