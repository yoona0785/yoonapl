# 宝宝档案优化需求文档

---

## 1. 业务背景

### 1.1 需求描述

2.17 版本需在现有宝宝档案原型基础上完成表单字段 v2.17 调整，以及标签/用户群宝宝档案指标同步更新。

### 1.3 目标结果

1. 装修页-表单组件字段按 v2.17 表格完成字段新增、字段名和字段说明更新。
2. 用户中心-标签、用户中心-用户群可使用宝宝档案字段作为指标圈选用户，并按v2.17 表格完成指标字段新增、调整

### 1.4 本期范围

1. 页面装修-表单组件-表单字段 v2.17 调整。
2. 用户中心-标签的新建/编辑页宝宝档案指标选择与条件配置。
3. 用户中心-用户群的新建/编辑页宝宝档案指标选择与条件配置。
4. 用户中心-用户管理和用户详情中宝宝信息的查看入口与编辑入口。
5. 小程序前端宝宝档案入口和填写后展示效果，用于表达后台配置后的前端承接样式。

### 1.5 本期不做

1. 不做真实支付、订单、积分、优惠券、营销发放等运营动作。

## 2. 关键业务规则

### 4.1 字段定义

#### 4.1.1 PC 后台菜单

| 一级菜单 | 二级菜单 | 说明 |
| --- | --- | --- |
| 商城装修 | 页面装修 | 页面装修列表页；点击新增或编辑进入装修编辑页 |
| 用户中心 | 用户管理 | 用户列表；点击用户昵称进入用户详情 |
| 用户中心 | 标签 | 标签列表；点击新增标签或编辑进入标签新建/编辑页 |
| 用户中心 | 用户群 | 用户群列表；点击新建用户群或编辑进入用户群新建/编辑页 |
| 系统配置 | 业务配置 | 平台业务配置，包含宝宝信息配置 |
| 小程序前端 | 无二级菜单 | 点击一级菜单直接打开小程序前端宝宝档案页面 |

#### 4.1.2 新增字段弹窗

1. 点击表单字段区域的添加按钮打开「添加字段」弹窗。
2. 字段按一级分类分组展示。
3. 一级分类按分类前编码从小到大排序，例如 `01-基础信息` 排在 `02-过敏档案` 前。
4. 分组标题展示时去掉编码，仅展示「基础信息」「过敏档案」等名称。
5. 点击字段后关闭弹窗，并将字段添加到当前表单字段列表和页面预览。

#### 4.1.3 字段显示逻辑

1. 弹窗标题：字段显示逻辑。
2. 可选字段范围：当前表单中字段类型为数字、日期、单选、多选、级联选择的字段。
3. 条件关系：支持所有条件、任一条件。
4. 条件操作符：
   1. 单选、多选、级联选择：等于、包含、不包含。
   2. 数字：等于、不等于、大于、大于等于、小于、小于等于、闭区间、开区间。
   3. 日期：等于、不等于、大于、大于等于、小于、小于等于、闭区间、开区间。
5. 字段填写值：
   1. 单选、多选取字段的选项值。
   2. 数字使用数字输入框；区间条件展示 2 个数字输入框。
   3. 日期使用日期选择框；区间条件展示 2 个日期选择框。

#### 4.1.5 用户中心-标签 / 用户群指标

1. 指标选择区增加「宝宝档案」大类。
2. 指标大类取表格一级分类，展示时去掉 `01-`、`02-` 等数字编码。
3. 指标分类取表格二级分类，展示时去掉 `01-`、`02-` 等数字编码。
4. 一级分类和二级分类均按数字编码从小到大排序。
5. 指标名称取表格指标名称。
6. 指标条件下拉取表格条件定义；条件定义为空时不展示条件下拉。
7. 指标值取表格指标值：
   1. 单选：只能选择 1 个值。
   2. 多选：可选择多个值。
   3. 非区间条件：展示 1 个输入框或选择框。
   4. 区间条件：展示 2 个输入框或选择框。
