# 页面装修跳转与企业微信配置需求文档

---

## 1. 业务背景

### 1.1 需求描述

1. 为了让用户在商城页面中看到更个性化的服务信息，页面装修组件需要支持引用用户已填写的宝宝档案、企业微信。
2. 用户添加的企业微信需要在页面上展示企微名称，并支持按配置打开企微链接。

### 1.3 目标结果

1. 运营选择“数据引用”后，可选择宝宝档案、企业微信。
2. 前台页面可根据当前登录用户展示对应的宝宝档案、企业微信名称。
3. 用户点击已配置数据跳转的展示区域后，可打开企微链接、小程序路径或 H5 链接。
4. 运营可在系统配置-内容配置中维护企业微信配置，并供页面装修引用。

### 1.4 本期范围

1. 商城装修-页面装修-组件配置：
   1.1 跳转链接“固定页面”类型新增企业微信，点击根据添加的企业微信打开不同的企微。
   1.2 新增宝宝档案数据预览样式。

2. 系统配置-内容配置：
   2.1 新增内容配置类型“企业微信”。
   2.2 企业微信名称只能下拉选择，不能手动输入。

3. 需要处理数据为空时的最终前台兜底展示策略。


### 1.5 本期不做

1. 不新增宝宝档案字段采集能力，仅引用既有宝宝档案数据。
2. 不新增企业微信成员管理能力，仅在内容配置中引用系统已有可选企微名单。
3. 不改造页面链接原有选择能力，页面链接沿用当前既有逻辑。

---

## 2. 关键业务规则

### 4.1 唯一性 / 并发性

1. 企业微信配置保存时，同一企微只能填写一次。

### 4.2 字段定义

1. 跳转链接：固定页面、数据引用
   - 是否必填：否

2. 固定页面：企业微信
   - 是否必填：当跳转链接=固定页面时必填
   - 默认值：空
   - 企微跳转：选择后根据添加的企微打开对应的企微链接，添加企微没有对应的企微链接或者没有添加企微点击跳转到企微客服配置的链接，若没有配置则无跳转

3. 数据模块：数据引用的数据来源。
   - 是否必填：当跳转链接=数据引用时必填
   - 默认值：空
   - 范围 / 长度：宝宝信息（性别、姓名、头像、生日）、宝宝档案、企业微信、上次消费时间、邀请码

4. 数据跳转：数据展示区域的点击跳转配置
   - 是否必填：否
   - 默认值：空
   - 范围 / 长度：可选企微链接、小程序路径、H5链接
   - 企微跳转：选择后根据添加的企微打开对应的企微链接，添加企微没有对应的企微链接或者没有添加企微点击跳转到企微客服配置的链接，若没有配置则无跳转

5. 宝宝档案字段：宝宝档案可引用字段集合
   - 是否必填：当数据模块=宝宝档案时至少选择一项
   - 默认值：空
   - 范围 / 长度：支持基础信息、过敏健康、喂养情况、肠道与营养、生长数据等字段

6. 企业ID：企业微信配置中的企微
   - 是否必填：企业微信配置保存时必填
   - 默认值：空
  

7. 企微名称：企业微信配置中的客服名称
   - 是否必填：企业微信配置保存时必填
   - 默认值：空
   - 范围 / 长度：来源于系统可用企业微信/企微成员名单，对接企微获取只能选择

8. 企微链接：企业微信配置中的会话链接
   - 是否必填：企业微信配置保存时必填
   - 默认值：空
   - 范围 / 长度：必须以 `http://` 或 `https://` 开头
   - 不允许：空值或非法链接格式

### 4.3 计算规则

1. 宝宝档案概要：
   - 规则：前台展示“宝宝档案概要”卡片，包含添加的宝宝档案的字段
   - 示例：宝宝月龄、当前奶粉品牌、当前辅食阶段、当前转奶进度、是否建议评估。

2. 企业微信名称匹配：
   - 规则：页面装修中不再选择具体企业微信，前台根据当前用户已添加的企业微信匹配企微名称。
   - 示例：用户添加“母婴顾问小安”，前台展示“母婴顾问小安”，添加多个前台只展示最近添加的企微
   - 边界说明：用户未添加企业微信前台展示“暂无顾问”点击跳转到企微客服配置的链接，若没有配置则无跳转

3. 内容配置列表企业微信摘要：
   - 规则：按企业微信配置顺序展示企微名称，多个名称使用顿号分隔。
   - 示例：`母婴顾问小安、营养师小禾`。
   - 边界说明：无配置时展示 `-`。

### 4.4 关键联动逻辑

1. 跳转链接联动：
   1.1 选择“数据引用”时，展示数据模块配置。
   1.2 选择“固定页面”时，展示固定页面配置。

2. 数据模块联动：
   2.1 选择“宝宝档案”时，展示宝宝档案字段选择、宝宝档案概要预览，并展示统一数据跳转配置。
   2.3 选择“企业微信”时，展示企微名称预览，并展示统一数据跳转配置。

3. 数据跳转联动：
   3.1 未选择数据跳转类型时，不展示跳转地址输入框。
   3.2 选择“小程序路径”时，跳转地址 label 为“小程序路径”，提示输入小程序页面路径。
   3.2 选择“H5链接”时，跳转地址 label 为“H5链接”，提示输入 H5 链接。

4. 宝宝档案字段选择联动：
   4.1 点击“选择字段”打开字段选择弹窗。
   4.2 搜索字段时，按字段名称模糊匹配。
   4.3 点击“全选当前结果”仅选择当前搜索结果。
   4.4 点击“清空”取消全部临时选择。
   4.5 点击“确认”后，将已选字段回填到配置区。

5. 企业微信配置联动：
   5.1 点击“添加企业微信”新增一条配置。
   5.2 点击单项垃圾桶图标时，弹出删除确认。
   5.3 点击“全部删除”时，弹出全部删除确认。
   5.4 新增、删除、全部删除均需点击“保存”后持久化。

### 4.5 异常处理规则

1. 跳转链接=数据引用但未选择数据模块：阻断发布，提示“请选择数据模块”。
2. 跳转链接=固定页面但未选择固定页面：阻断发布，提示“请选择固定页面”。
3. 数据模块=宝宝档案但未选择宝宝档案字段：阻断发布，提示“请至少选择一个宝宝档案字段”。
4. 已选择数据跳转类型但未填写跳转地址：阻断发布，提示“请填写{数据跳转类型}”。
5. 企微链接或 H5 链接未以 `http://` 或 `https://` 开头：阻断发布或保存，提示“{数据跳转类型}需以 http:// 或 https:// 开头”。
6. 企业微信配置保存时，任一记录缺少企微名称或企微链接：阻断保存，提示“请选择企微名称并填写企微链接”。
7. 企业微信配置为空时不允许保存。
8. 前台宝宝档案数据为空时，展示提示“完善宝宝档案”，点击跳转到 业务配置-宝宝信息-填写页
9. 前台企微数据为空时，展示“暂无顾问”，点击跳转到企微客服配置的链接，若没有配置则无跳转

### 4.6 权限规则（按需）

1. 页面装修配置权限沿用“商城装修-页面装修”现有权限。
2. 内容配置企业微信权限沿用“系统配置-内容配置”现有权限。

### 4.7 数据来源（按需）

1. 宝宝档案：来源于当前用户已填写的宝宝档案。
2. 企业微信：来源于用户已添加企业微信与系统配置-内容配置中的企业微信配置。
3. 固定页面：来源于系统预置页面枚举。

### 4.8 通知机制（按需）

1. 保存草稿成功：Toast 提示“草稿已保存”。
2. 页面发布成功：Toast 提示“页面发布成功”。
3. 宝宝档案字段保存成功：Toast 提示“宝宝档案字段已更新”。
4. 企业微信配置保存成功：Toast 提示“企业微信配置已保存”。
5. 单项删除成功：Toast 提示“已删除”。
6. 全部删除成功：Toast 提示“已全部删除”。

### 4.10 上限与极限（按需）

1. 宝宝档案字段选择数量上限：无限制
2. 企业微信配置条数上限：无限制

---

## 5. 状态与异常

### 5.1 空态

1. 宝宝档案为空：展示提示“完善宝宝档案”，点击跳转到 业务配置-宝宝信息-填写页
2. 企业微信未匹配到用户已添加企微：展示“暂无顾问”，点击跳转到企微客服配置的链接，若没有配置则无跳转
3. 企业微信配置为空：内容配置列表内容列展示 `-`。
