订单兑换列表

页面路径

营销中心 - 订单兑换

流程状态

说明：订单兑换活动按活动时间和人工操作自动流转状态。

- 待开始：编辑、复制、删除、活动店铺、推广、日志

- 进行中：查看详情、复制、立即结束、活动店铺、推广、日志

- 已结束：查看详情、复制、推广、日志

列表逻辑

活动数据

1. 兑换用户数：统计当前活动下成功兑换或审核通过后发放奖励的去重用户数，按用户 ID 去重 count，展示在活动数据卡片。

2. 兑换订单数：统计当前活动下成功兑换或审核通过后发放奖励的订单数，按兑换记录 count。

3. 数据更新时间：展示活动数据最近统计时间，格式 YYYY.MM.DD HH:mm:ss，点击「刷新」后重新计算当前活动统计。

表搜索

1. 活动名称：文本输入-模糊匹配，匹配活动名称字段，支持空值查询。

2. 活动 ID：文本输入-精确匹配，匹配活动 ID。

3. 活动类型：下拉单选，枚举值为全部、订单金额固定积分、订单金额比例积分、商品数量固定积分、商品数量比例积分。

4. 活动状态：下拉单选，枚举值为全部、待开始、进行中、已结束。

5. 活动时间：日期时间范围，匹配活动开始时间和结束时间。

多字段间搜索关系：AND（同时满足）

查询表

排序与分页：默认按创建时间倒序排列，默认 10 条/每页，支持分页切换。

1. 活动信息（多信息字段）

- 1.1 活动名称：取活动名称，蓝色常规体展示，点击进入活动详情页；详情页复用订单兑换新增/编辑页，只读不可编辑。

- 1.2 活动 ID：取活动 ID，展示在活动名称下方。

- 1.3 活动类型：取活动类型，标签样式展示。

2. 活动时间：展示开始时间和结束时间；长期有效活动展示「长期有效」。

3. 兑换用户：展示兑换用户数，点击进入兑换数据页，并默认按当前活动名称筛选。

4. 兑换订单：展示兑换订单数。

5. 活动状态：枚举值说明。

- 待开始：橙色状态标签。

- 进行中：绿色状态标签。

- 已结束：灰色状态标签。

6. 创建时间：展示创建人和创建时间，时间格式 YYYY-MM-DD HH:mm:ss。

7. 更新时间：展示更新人和更新时间，时间格式 YYYY-MM-DD HH:mm:ss。

8. 操作列：按活动状态和兑换维度展示操作。

- 待开始：编辑、复制、删除、活动店铺、推广、日志。

- 进行中：复制、立即结束、活动店铺、推广、日志。

- 已结束：复制、推广、日志。

- 活动店铺：仅活动状态为待开始/进行中，且兑换维度为店铺维度或店铺产品组维度时展示。

列表交互

1. 兑换审核：点击按钮，跳转兑换审核列表页，打开独立页签。

2. 公域店铺：点击按钮，跳转公域店铺列表页，打开独立页签。

3. 订单限制：点击按钮，跳转订单限制页，打开独立页签。

4. 新增：点击按钮，跳转订单兑换新增页，打开独立页签。

5. 活动名称：点击活动名称，进入订单兑换详情页，页面字段全部只读不可编辑。

6. 兑换用户：点击兑换用户数，进入兑换数据页，并自动带入当前活动名称作为筛选条件。

7. 删除：点击删除，展示二次确认弹窗。

- 标题：删除

- 内容：是否确认删除

- 确认后删除该活动，刷新列表。

8. 立即结束：点击立即结束，展示二次确认弹窗。

- 标题：立即结束

- 内容：是否立即结束，结束后用户将不可领取

- 确认后将活动状态更新为已结束，结束时间更新为当前时间。

9. 活动店铺：点击后跳转活动店铺页，携带活动 ID、活动名称和兑换维度。

10. 推广：点击后打开推广弹窗。

- 小程序链接与二维码：展示小程序链接、小程序路径、海报图，支持复制链接、下载小程序码、下载海报。

- 小程序分享设置：支持填写分享标题、上传分享图和海报，保存后作为活动推广配置。

11. 日志：点击后打开操作日志弹窗，字段包含日志 ID、操作人、操作描述、提示信息、操作时间。

订单兑换新增/编辑

页面路径

营销中心 - 订单兑换 - 订单兑换新增/编辑

表单逻辑

基本信息

1. 活动类型：单选组，必填，枚举值为订单金额固定积分、订单金额比例积分、商品数量固定积分、商品数量比例积分。

2. 活动名称：文本输入，必填，限制 50 字符内。

3. 活动时间：日期时间范围，选填，支持长期有效；未勾选长期有效时可选择开始时间和结束时间，精确到秒。

4. 活动背景图：图片上传，必填，取上传图片作为小程序订单兑换首页背景图，按原图比例等比展示。

5. 活动规则：多行文本，选填，有配置时小程序背景图右侧展示活动规则入口，点击后弹窗展示完整规则并支持滑动。

6. 积分抵扣比例：文本输入，选填，用于小程序兑换成功弹窗展示，如 100积分=1元。

7. 数据播报：开关，选填，开启后小程序首页展示兑换播报。

规则设置

8. 兑换积分：单选组，必填，枚举值为固定积分、比例积分。

9. 兑换订单范围：日期时间选择，选填，提示为订单完成开始时间；配置后小程序展示 YYYY.MM.DD HH:mm:ss后完成的订单，未配置不展示。

10. 兑换维度：单选组，必填，枚举值为统一维度、店铺维度、店铺产品组维度。

11. 选择店铺：按钮，条件展示。

- 11.1 统一维度：不展示选择店铺。

- 11.2 店铺维度：展示选择店铺按钮，可选择公域店铺。

- 11.3 店铺产品组维度：展示选择店铺按钮，按产品组维度选择，仅启用中的产品组可选择；没有产品组或产品组均禁用的店铺不可选择。

12. 活动积分：按兑换维度展示。

- 12.1 统一维度：展示一行规则表达，格式为 订单满/订单每满/商品满/商品每满 + 门槛值 + 单位 + 可兑 + 积分值 + 积分。

- 12.2 店铺维度：展示店铺维度表格，字段为操作、店铺、活动积分；支持批量设置和逐行填写。

- 12.3 店铺产品组维度：展示平铺表格，字段为操作、店铺、产品组、活动积分；不按店铺分组，不展示商品 ID 字段；支持批量设置和逐行填写。

推广分享

13. 小程序分享标题：文本输入，选填，限制 50 字符内。

14. 分享图：图片上传，选填，用于微信小程序分享卡片预览。

15. 海报：图片上传，选填，用于推广弹窗海报展示。

列表交互

1. 保存：点击保存按钮。

- 1.1 校验必填字段，未填写时在字段下方展示错误提示。

- 1.2 校验活动时间，同一个时间段只能创建一个活动，活动开始时间或结束时间不能与已有活动重复，时间范围不能与已有活动重叠。

- 1.3 校验通过后保存到活动列表，新增后返回订单兑换列表，编辑后更新列表和活动店铺数据。

2. 取消：点击取消按钮，返回订单兑换列表；如有未保存内容，展示离开确认。

3. 选择店铺：点击打开选择店铺弹窗。

- 3.1 店铺维度弹窗字段为店铺、创建时间；店铺展示平台标签、店铺名称、ID。

- 3.2 店铺产品组维度弹窗字段为店铺、产品组、创建时间；产品组只展示产品组名称，不展示商品 ID 数量和启用/禁用提示。

- 3.3 已选择项置灰勾选不可再次选择。

- 3.4 支持全选全部可选择项，确认后最新添加的店铺或产品组显示在前。

4. 批量设置：输入批量门槛和批量积分后点击批量设置，将值写入当前配置表所有可编辑行。

5. 删除：点击表格行删除，删除当前店铺或产品组配置；删除后该项可重新在选择店铺弹窗中选择。

活动店铺

页面路径

营销中心 - 订单兑换 - 活动店铺

列表逻辑

表搜索

1. 店铺名称：文本输入-模糊匹配，匹配店铺名称。

2. 店铺 ID：文本输入-精确匹配，匹配店铺 ID。

3. 平台：下拉单选，枚举值为全部、淘宝、天猫、京东、抖音、快手、拼多多、小红书、其他。

多字段间搜索关系：AND（同时满足）

查询表

排序与分页：默认按活动中添加顺序展示，分页展示。

1. 店铺维度表格字段：

- 1.1 店铺：展示平台标签、店铺名称、店铺 ID。

- 1.2 活动积分：展示当前店铺积分规则，可编辑门槛值和积分值。

- 1.3 添加时间：展示添加人和添加时间，格式 YYYY-MM-DD HH:mm:ss。

- 1.4 操作：删除。

2. 店铺产品组维度表格字段：

- 2.1 操作：删除产品组。

- 2.2 店铺：展示平台标签、店铺名称、店铺 ID。

- 2.3 产品组：展示产品组名称。

- 2.4 活动积分：展示当前产品组积分规则，可编辑门槛值和积分值。

- 2.5 添加时间：展示添加人和添加时间。

列表交互

1. 添加店铺：点击打开选择店铺弹窗，规则与订单兑换新增/编辑页一致。

2. 批量设置：点击后将批量门槛和批量积分写入当前列表全部行。

3. 删除：点击删除后移除当前店铺或产品组配置，页面标记为未保存。

4. 保存：点击保存，将当前活动店铺配置写回订单兑换活动数据；保存成功后 Toast 提示「保存成功，活动店铺已生效」。

5. 取消：点击取消返回订单兑换列表；未保存内容不生效。

页面路径

列表逻辑

表搜索

2. 订单号：文本输入-精确匹配，匹配兑换订单号。

3. 活动状态：下拉单选，枚举值为全部、待开始、进行中、已结束。

4. 兑换时间：日期时间范围，匹配兑换创建时间或积分发放时间。

排序与分页：默认按兑换时间倒序排列。

1. 活动信息：展示活动名称、活动 ID、活动状态。

2. 兑换用户：展示用户昵称、手机号、用户 ID。

3. 兑换订单：展示兑换订单号，点击打开订单详情弹窗。

4. 兑换积分：仅兑换成功记录展示积分值。

- 1.1 订单信息字段包含兑换订单、平台用户 ID、订单状态、订单实付金额、贸易模式、店铺编码、店铺名称、出库时间、下单时间、创建时间、支付时间、签收时间、完成时间。

- 1.2 商品信息字段包含商品编码、商品名称、成交单价、购买数量、活动折扣、优惠券、税费、运费、尾差金额、商品总金额。

2. 订单兑换标签：点击顶部订单兑换页签返回订单兑换列表。

兑换审核

页面路径

营销中心 - 订单兑换 - 兑换审核

流程状态

- 待审核：审核

- 通过：查看

- 驳回：查看

列表逻辑

表搜索

1. 用户信息：文本输入-模糊匹配，匹配用户昵称、手机号或用户 ID。

2. 活动名称：文本输入-模糊匹配，匹配活动名称。

3. 订单号：文本输入-精确匹配，匹配提交的兑换订单号。

4. 审核状态：下拉单选，枚举值为全部、待审核、通过、驳回。

查询表

排序与分页：默认按提交时间倒序排列。

1. 用户信息：展示昵称、手机号、用户 ID。

2. 活动信息：展示活动名称、活动 ID。

3. 提交信息：展示下单平台、店铺名称、截图、补充说明。

4. 审核状态：展示顺序为状态、积分、审核意见。

- 待审核：状态展示待审核。

- 通过：状态展示通过，展示兑换积分。

- 驳回：状态展示兑换失败，展示审核意见。

5. 提交时间：格式 YYYY-MM-DD HH:mm:ss。

6. 操作：待审核展示审核，已审核展示查看。

列表交互

1. 审核：点击打开审核弹窗。

- 1.1 审核信息字段：审核结果、兑换积分、审核意见。

- 1.2 提交信息字段：下单平台、店铺名称、截图、补充说明、用户信息。

- 1.3 审核通过后，将对应兑换记录更新为兑换成功并发放积分。

- 1.4 审核驳回后，将对应兑换记录更新为兑换失败并记录审核意见。

2. 查看：点击打开只读审核详情弹窗。

公域店铺新增/编辑

页面路径

营销中心 - 订单兑换 - 公域店铺 - 店铺新建/编辑

表单逻辑

基础信息

1. 电商平台：单选组，必填，枚举值为淘宝、天猫、京东、抖音、快手、拼多多、小红书、其他。

2. 店铺名称：文本输入，必填，限制 50 字符内。

3. 店铺 ID：文本输入，必填，同平台下不可重复。

产品组配置

4. 产品组名称：文本输入，必填。

5. 商品 ID：多行文本，必填，多个商品 ID 支持换行输入；同一店铺下同一商品 ID 不可归属多个产品组。

6. 默认积分：数字输入，选填，正整数，用于订单兑换活动按店铺-产品组维度配置时作为默认积分值。

7. 产品组状态：开关，默认启用，可在编辑页切换启用/禁用。

列表交互

1. 新增产品组：点击打开产品组弹窗，填写产品组名称、商品 ID、默认积分。

2. 编辑产品组：点击编辑回填产品组数据。

3. 删除产品组：点击删除移除产品组；若已被活动引用，需提示不可删除或待确认处理。

4. 保存：校验通过后保存店铺及产品组数据，返回公域店铺列表。

5. 取消：返回公域店铺列表，未保存内容不生效。
