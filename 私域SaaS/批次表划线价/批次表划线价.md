# 需求文档：批次表、划线价、商详配置

---

## 1. 业务背景

### 1.1 需求描述

商品详情页存在评价、问答模块，不同平台运营能力不同，需要后台支持按平台运营策略控制是否开放。

商品可能不参与任何活动直接低价售卖，需要在商品管理维护划线价，并在小程序商品列表、商品详情、购物车、提单页按统一口径展示。

商品关联中台同步批次，需要新增批次管理能力，支持运营查看批次基础信息、价格、库存、状态、价格趋势，并支持批次在商城前台的上下架控制、导出和日志追踪。

### 1.3 目标结果

- 运营可在系统配置中控制商品详情是否展示商品评价、商品问答。
- 商品新增/编辑可维护计量单位和 SKU 维度划线价，前台按有效划线价展示删除线价格。
- 批次管理可查询、查看详情、上下架、导出批次数据，并记录关键操作日志。
- 前台小程序商品列表、商品详情、选规格、购物车、提单页、订单按商品类型、活动类型、划线价和计量单位展示。

### 1.4 本期范围

- 系统配置-业务配置：保留已有配置，新增商详配置分组。
- 商品中心-商品管理：列表、新增/编辑、导出补充划线价和计量单位相关能力。导入修改支持划线价和计量单位导入
- 商品中心-批次管理：新增批次列表、搜索、导出、详情、上下架、日志、备案信息、报关信息、商品详情图查看。
- 微信小程序前台展示规则：商品列表、商品详情、选规格、购物车、提单页的价格、划线价、评价问答、计量单位展示规则。

### 1.5 本期不做

- 不新增商品评价、商品问答内容运营后台。
- 不新增活动创建、秒杀、折扣立减配置能力，仅定义前台价格展示口径。
- 不改造商品属性、分类的计量单位继承逻辑；本期计量单位直接在商品管理维护。
- 不改变中台批次状态，只控制商城状态。

## 2. 关键业务规则

### 4.1 唯一性 / 并发性

- 商品 ID、API 商品编码、批次号、物料编码、供应商编码由对应业务系统或中台提供，不允许后台手工重复创建相同主键数据。
- 批次号作为批次管理主识别字段，同一批次号在列表中仅展示一条批次主记录；同一批次下可包含多条规格价格记录。
- 批次商城状态上下架存在并发操作时，以最后一次成功提交为准，并记录操作日志。
- 商品编辑保存时如商品已被其他人更新，建议后端返回冲突提示并要求刷新后重试。

### 4.2 字段定义

#### 系统配置-业务配置

- 商品评价：控制商品详情评价模块展示。
  - 是否必填：是
  - 默认值：开启
  - 选项：开启、关闭
  - 备注：关闭后商品详情隐藏评价入口
- 商品问答：控制商品详情问答模块展示。
  - 是否必填：是
  - 默认值：开启
  - 选项：开启、关闭
  - 备注：关闭后商品详情隐藏问答入口

#### 商品中心-商品管理-列表

- 商品导出：增加划线价、计量单位
- 导入修改：增加划线价、计量单位

#### 商品中心-商品管理-新增/编辑

- 计量单位：商品规格展示使用的单位。
  - 是否必填：否
  - 默认值：个
  - 范围 / 长度：10 字内
  - 不允许：超过 10 字

- 价格限购：维护成本价、售价、划线价、重量、限购。
  - 成本价：必填，金额
  - 售价：必填，金额
  - 划线价：非必填，金额，填写需大于售价
  - 重量：必填，单位 kg
  - 限购：非必填，正整数


#### 商品中心-批次管理-列表

- 搜索字段：批次号、物料编码、商品 ID、商品名称、供应商名称、分发中台、贸易模式、经营模式、中台状态、商城状态、价格趋势。
- 批次号、物料编码、商品 ID 支持批量查询，多个值用英文逗号隔开。
- 价格趋势选项：涨价、降价、未调价。
- 列表字段：商品名称、关联商品、批次号、供应商名称、供应商库位编码、分发中台、效期、贸易模式、经营模式、已用库存、剩余库存、价格趋势、中台状态、商城状态、更新时间、操作。
- 商品名称：展示商品图片、商品名称、物料编码。
- 关联商品：展示商品 ID；如关联多个商品 ID，一行一个商品 ID，点击商品 ID 进入后台商品查看页。超过3个可以收起展开
- 供应商名称：展示供应商名称和供应商编码。
- 供应商库位编码：展示该供应商在中台/库位维度的编码。
- 操作：上架/下架、日志。

#### 商品中心-批次详情

- 商品信息：展示商品图、商品名称、批次号、物料编码、商品详情、商品品牌、分类、供应商、供应商编码、经营模式、贸易模式、备案信息、报关信息、中台状态、商城状态。
- 商品详情：点击查看，打开商品详情图弹窗。
- 备案信息：点击查看，打开备案信息弹窗；字段为 HSCode、关税税率、增值税率、消费税率。
- 报关信息：点击查看，打开报关信息弹窗；字段为关区、报关单、放行通知书、检验检疫报告。
- 价格信息：展示规格、总规格毛重(kg)、商品单价、商品总价、起购、限购。
- 中台价格趋势：展示同步前价格、同步后价格、趋势、同步时间。
- 邮费信息：按区域展示首重(kg)、首费(元)、续重(kg)、续费(元)。
- 页面操作：日志、上架/下架。

### 4.3 计算规则

- 商品划线价有效规则：划线价有值且划线价大于售价时有效；划线价为空、不填、等于售价或小于售价时前台不展示划线价。
- 商品保存校验：划线价有值且小于或等于售价时，保存失败并提示“划线价需大于售价”。
- 商品列表价格：展示商品当前售价；导出时规格信息需包含划线价字段。
- 前台规格计量单位：规格展示文案 = 规格数量 + 商品计量单位 + 装。示例：计量单位为“个”时，展示“1个装”。
- 批次价格趋势：
  - 批次规格1同步后价格高于同步前价格，列表显示“涨价”。
  - 批次规格1同步后价格低于同步前价格，列表显示“降价”。
  - 所有规格价格无变化，列表显示“-”，筛选时归类为“未调价”。

### 4.4 关键联动逻辑

- 商品评价关闭后，商品详情页隐藏评价入口、评价列表、晒图等评价模块。
- 商品问答关闭后，商品详情页隐藏问答入口和问一问模块。
- 商品管理维护计量单位后，小程序商品详情规格 Tab、加入购物车规格弹层、立即购买规格弹层、订单列表、订单详情的规格均读取该计量单位。
- 商品管理维护划线价后，小程序商品列表、商品详情、购物车推荐商品等按有效划线价展示删除线价格。
- 批次上下架只改变商城状态；中台状态仍以中台同步结果为准。
- 批次关联商品 ID 点击后，如后台存在商品详情数据，则进入商品查看/编辑页；如不存在，给出暂无本地详情提示。
- 批次详情中备案信息、报关信息、商品详情图均通过弹窗展示，不离开当前批次详情页。

### 4.5 异常处理规则

- 商品新增/编辑必填项为空时不可保存，并提示对应字段必填。
- 批次列表无匹配数据时展示“暂无匹配数据”。
- 批次上下架失败时保持原商城状态，并提示失败原因。
- 批次导出结果为空时提示无可导出数据。
- 批次详情数据不存在时返回批次列表并提示批次不存在或已删除。
- 弹窗加载备案信息、报关信息失败时展示失败提示并允许重试。
- 小程序端划线价无效时仅展示当前价格，不展示空白占位。
- 小程序端评价、问答关闭时隐藏对应模块，不展示关闭提示。

### 4.6 权限规则（按需）

- 批次管理：具备批次管理权限的账号可查询、查看详情、导出、上架/下架、查看日志。

### 4.7 时效与有效期（按需）

- 批次效期：以中台同步批次效期为准，在批次列表和批次详情展示。
- 中台价格趋势同步时间：以中台最近一次同步时间或批次更新时间为准。

### 4.8 上限与极限（按需）

- 计量单位：5 字内。
- 批次号、物料编码、商品 ID 批量查询：多个值用英文逗号隔开；单次最大查询数量500。
- 批次导出：导出当前筛选结果；单次最大导出条数无限制。

## 5. 状态与异常

### 5.1 业务状态

- 批次中台状态：上架、停售。
- 批次商城状态：上架、下架。
- 批次价格趋势：涨价、降价、未调价。
- 商品评价配置：开启、关闭。
- 商品问答配置：开启、关闭。

### 5.2 页面默认态

- 商品新增计量单位默认为“个”。
- 批次管理默认展示批次列表，更多搜索收起，仅展示首行搜索字段。
- 批次详情默认展示商品信息、价格信息、中台价格趋势、邮费信息。

### 5.3 空态

- 批次管理：查询无结果时展示“暂无匹配数据”。
- 日志弹窗：无日志时展示“暂无日志”。

### 5.4 异常态 / 失败态

- 保存配置失败：提示失败原因，页面保留当前填写内容。
- 商品保存失败：提示字段校验结果，保留当前填写内容。
- 划线价不合法：提示“划线价需大于售价”。
- 商品导出失败：提示导出失败。
- 批次上下架失败：提示失败原因，商城状态不变。
- 批次导出失败：提示导出失败。
- 弹窗数据加载失败：展示失败提示并允许关闭后重试。

### 5.6 状态-操作矩阵（按需）

| 对象 | 状态 | 可执行操作 | 备注 |
| --- | --- | --- | --- |
| 批次 | 商城上架 | 下架、日志、查看详情、导出 | 下架只改变商城状态 |
| 批次 | 商城下架 | 上架、日志、查看详情、导出 | 上架前需二次确认 |
| 批次 | 中台上架 | 可按权限操作商城上下架 | 不改变中台状态 |
| 批次 | 中台停售 | 日志、查看详情、导出 | 禁用商城上架 |

## 6. 衍生需求与默认授权项

### 6.1 导出

- 商品导出文件包含商品信息和规格信息；规格信息必须包含划线价字段。商品信息包含计量单位
- 批次管理支持按当前筛选结果导出。
- 批次导出字段：商品名称、物料编码、批次号、供应商名称、供应商编码、供应商库位编码、分发中台、效期、贸易模式、经营模式、已用库存、剩余库存、中台状态、商城状态、规格、规格毛重、商品单价、商品总价、起购、限购、价格趋势。
- 批次导出建议文件名：批次管理导出_{时间戳}.xls / .xlsx。
- 导出字段顺序需与需求字段顺序保持一致。

### 6.2 日志

- 批次管理需记录：商城上架、商城下架、中台价格同步、关联商品变化、导出操作。待确认：导出是否记录日志。
- 批次日志字段：日志 ID、操作人、操作描述、提示信息、操作时间。
- 操作记录入口命名统一为“日志”。

### 6.3 ID 规则

- 批次号：中台同步，作为批次查看和导出主字段。
- 物料编码：中台同步，列表商品名称列下展示。
- 供应商编码：供应商主数据提供。
- 供应商库位编码：中台/库位维度提供。
- 日志 ID：系统生成，需保证唯一。
