# 用户限购及内容运营需求文档

## 1. 业务背景

### 1.1 需求描述
部分商品因品类、价格、合规或运营策略，需要对特定用户隐藏不可见；同时部分商品需要限制用户购买数量，避免超量购买、异常囤货。

运营侧还需要在用户管理列表批量导入积分，以及在内容中心商品评价列表批量导入评价，用于补充运营数据、提升后台运营效率。

### 1.3 目标结果
- 支持配置商品屏蔽规则：命中用户在列表、搜索等入口不可见相关商品；若通过链接进入商品详情，提示商品已下架。
- 支持配置商品限购规则：命中用户可见商品，但下单时需按限购规则校验购买数量。
- 支持在用户管理列表导入积分，并提供模板下载、文件上传、解析校验、保存导入。
- 支持在商品评价列表导入评价，并提供模板下载、文件上传、解析校验、保存导入。

### 1.4 本期范围
- 用户中心-用户管理：新增导入积分入口。
- 用户中心-用户限购：新增商品屏蔽、商品限购配置能力。
- 内容中心-商品评价：新增导入评价入口，并补充评价来源筛选和评价来源标签。
- 用户限购支持列表、新建、编辑、启停、日志、删除。
- 商品屏蔽/商品限购均支持手动选择商品或按商品分类配置。
- 商品屏蔽人群支持指定人群、指定标签、指定用户。
- 商品限购人群支持全部、指定人群、指定标签、指定用户。
- 商品限购设置支持终身限购、按单限购、按周期限购；周期支持每天、每月、指定周期。
- 用户管理导入积分模板字段：用户ID、积分数量、变更说明。
- 商品评价导入评价模板字段：头像、昵称、商品ID、评价内容、评价图片、商品评分、物流配送、平台服务、评价区状态。

### 1.5 本期不做
- 不做商品屏蔽/限购规则审核流程。
- 不做积分导入后的财务审批流程。
- 不做评价导入后的审核流，仅按导入状态展示。

## 2. 关键业务规则

### 4.1 唯一性 / 并发性
- 规则名称在同一规则类型下建议唯一，若重复需提示运营确认，避免难以区分规则。
- 同一商品同时命中多条屏蔽规则时，只要任一启用规则命中用户，即对该用户不可见。
- 同一商品同时命中多条限购规则时，按更最小的可购买数量执行。
- 商品评价导入时，同一商品ID、昵称、评价内容重复不允许导入。

### 4.2 字段定义
- 用户限购规则名称
  - 是否必填：是。
  - 默认值：无。
  - 范围 / 长度：1-50 个字符。
  - 不允许：全空格。
- 商品范围
  - 是否必填：是。
  - 可选值：手动选择、商品分类。
  - 默认值：手动选择。
- 商品列表
  - 是否必填：当商品范围为手动选择时必填。
  - 交互：点击「选择商品」打开商品选择弹窗；选择后在表单内展示商品列表。
  - 商品列表支持上移、下移、删除。
- 商品分类
  - 是否必填：当商品范围为商品分类时必填。
  - 交互：使用下拉多选框，支持多选；选择后或点击空白处下拉收起。
- 适用人群
  - 商品屏蔽可选：指定人群、指定标签、指定用户。
  - 商品限购可选：全部、指定人群、指定标签、指定用户。
  - 默认值：商品屏蔽默认指定人群；商品限购默认全部。
- 指定人群 / 指定标签
  - 是否必填：当适用人群选择对应类型时必填。
  - 交互：下拉多选。
- 指定用户
  - 是否必填：当适用人群为指定用户时必填。
  - 输入方式：多行文本框，用户ID支持英文逗号、换行或空格分隔。
  - 支持点击「查看用户」打开用户信息弹窗，展示用户ID、用户昵称、手机号、性别、注册时间。
- 限购方式
  - 是否必填：商品限购必填。
  - 可选值：终身限购、按单限购、按周期限购。
- 限购数量
  - 是否必填：商品限购必填。
  - 范围：大于 0 的整数。
  - 按周期限购时，周期可选每天、每月、指定周期；指定周期需填写 N 天。
- 用户限购状态
  - 新建默认启用。
  - 新建/编辑表单不展示状态字段。
  - 状态仅在列表通过开关启停。
- 评价来源
  - 是否必填：系统生成。
  - 可选值：首次评价、追加评价、导入评价。
  - 展示位置：商品评价列表提交内容字段内，以标签形式展示。
  - 搜索：商品评价更多搜索中支持按评价来源筛选。
- 导入积分模板字段
  - 用户ID：必填。
  - 积分数量：必填，非 0 数字，正整数表示加积分，负整数表示减积分。
  - 变更说明：必填。
- 导入评价模板字段
  - 头像：非必填。
  - 昵称：必填。
  - 商品ID：必填。
  - 评价内容：必填。
  - 评价图片：非必填。
  - 商品评分：必填，1-5 分。
  - 物流配送：必填，1-5 分。
  - 平台服务：必填，1-5 分。
  - 评价区状态：必填，可选隐藏、展示。
- 创建时间 / 更新时间
  - 用户限购列表展示账号名称 + 时间。
  - 新建时记录创建账号、创建时间、更新账号、更新时间。
  - 编辑、启停时更新更新账号和更新时间。

### 4.3 计算规则
- 商品屏蔽
  - 用户命中启用状态的屏蔽规则后，相关商品在商品列表、搜索结果中不可见。
  - 若用户通过缓存、分享链接或历史记录进入商品详情，前端提示「商品已下架」。
- 商品限购
  - 终身限购：统计用户历史累计购买数量，累计数量 + 本次购买数量不得超过配置数量。
  - 按单限购：单笔订单内该商品购买数量不得超过配置数量。
  - 按周期限购：按每天、每月或指定周期统计购买数量，周期内累计数量 + 本次购买数量不得超过配置数量。
- 导入积分
  - 导入文件表头必须为：用户ID、积分数量、变更说明。
  - 积分数量需为非 0 数字。
  - 解析后统计总条数、成功条数、失败条数。
- 导入评价
  - 导入文件表头必须为：头像、昵称、商品ID、评价内容、评价图片、商品评分、物流配送、平台服务、评价区状态。
  - 商品评分、物流配送、平台服务必须为 1-5 分。
  - 后台导入评价的用户手机号、用户ID为空。
  - 后台导入评价的订单编号为「-」。
  - 商品信息根据商品ID从商品库获取，获取不到显示「-」。
  - 导入成功后评价来源固定为「导入评价」。

### 4.4 关键联动逻辑
- 用户限购列表 Tab 切换
  - 商品屏蔽 Tab 展示商品屏蔽规则。
  - 商品限购 Tab 展示商品限购规则。
- 用户限购新建 / 编辑
  - 从商品屏蔽 Tab 进入商品屏蔽表单。
  - 从商品限购 Tab 进入商品限购表单。
  - 表单内不支持切换规则类型。
- 商品范围切换
  - 切换到手动选择时展示商品列表。
  - 切换到商品分类时展示分类下拉多选。
- 适用人群切换
  - 切换到指定人群 / 指定标签时展示下拉多选。
  - 切换到指定用户时展示多行输入框和查看用户入口。
  - 商品限购选择全部时不展示额外选择控件。
- 用户管理导入积分
  - 点击「下载模板」下载 CSV 模板。
  - 点击上传区域选择文件并解析。
  - 解析后展示总条数、成功条数、失败条数和行级结果。
  - 全部校验通过后点击保存，提示导入成功。
- 商品评价导入评价
  - 点击「下载模板」下载 CSV 模板。
  - 点击上传区域选择文件并解析。
  - 解析后展示总条数、成功条数、失败条数和行级结果。
  - 全部校验通过后点击保存，新增评价记录并回到商品评价列表。
  - 新增记录展示评价来源「导入评价」。

### 4.5 异常处理规则
- 保存规则时，规则名称为空提示「请输入规则名称」。
- 手动选择商品为空提示「请选择商品」。
- 商品分类为空提示「请选择商品分类」。
- 指定人群、指定标签、指定用户为空提示「请选择适用人群」。
- 限购数量为空、为 0 或非数字提示「请输入正确的限购数量」。
- 导入积分未上传文件点击保存，提示「请先上传并解析文件」。
- 导入积分存在解析失败数据时，提示「存在解析失败数据，请修正后重新上传」。
- 导入评价未上传文件点击保存，提示「请先上传并解析文件」。
- 导入评价存在解析失败数据时，提示「存在解析失败数据，请修正后重新上传」。
- 删除规则需二次确认，确认文案为「是否确认删除」。

### 4.6 权限规则（按需）
- 用户限购菜单、规则新建、编辑、删除、启停、日志查看需按后台权限控制。
- 用户管理导入积分、下载模板需按运营权限控制。
- 商品评价导入评价、下载模板需按内容运营权限控制。
- 无权限时隐藏入口或置灰按钮，并提示无权限操作。

### 4.7 时效与有效期（按需）
- 用户限购规则不配置生效时间和失效时间。
- 启用后立即生效，停用后立即失效。
- 按周期限购的统计周期以服务端订单支付成功时间为准。
- 导入积分和导入评价保存成功后立即生效。

### 4.8 数据来源（按需）
- 商品数据来源：商品中心商品列表。
- 商品分类来源：商品中心分类管理。
- 人群数据来源：用户中心用户群。
- 标签数据来源：用户中心标签。
- 指定用户来源：用户中心用户管理。
- 积分导入用户校验来源：用户中心用户数据。
- 评价导入商品信息来源：商品库，按商品ID匹配。

### 4.9 上限与极限（按需）
- 单条规则手动选择商品数量无上限。
- 单条规则选择分类数量无上限。
- 指定用户一次输入数量无上限。
- 导入积分单次文件行数无上限。
- 导入评价单次文件行数无上限。
- 评价图片字段支持图片嵌入单元格。

## 5. 状态与异常

### 5.1 业务状态
- 用户限购规则状态：启用、停用。
- 商品评价来源：首次评价、追加评价、导入评价。

### 5.2 页面默认态
- 用户限购默认展示商品屏蔽 Tab。
- 商品屏蔽 / 商品限购列表默认展示规则名称、人群、状态、创建时间、更新时间、操作。
- 新建用户限购规则默认状态为启用。
- 导入积分弹窗默认展示下载模板和上传文件区域。
- 导入评价弹窗默认展示下载模板和上传文件区域。

### 5.3 空态
- 商品屏蔽 / 商品限购列表无规则时展示暂无数据。
- 导入积分文件无有效数据时展示解析失败。
- 导入评价文件无有效数据时展示解析失败。

### 5.4 异常态 / 失败态
- 列表筛选无结果展示空态。
- 导入积分表头不符合模板要求，展示解析失败，并提示表头需为「用户ID、积分数量、变更说明」。
- 导入积分行数据缺少必填项或积分数量不合法，展示行级失败。
- 导入评价表头不符合模板要求，展示解析失败。
- 导入评价行数据缺少必填项、评分不合法或评价区状态不合法，展示行级失败。
- 导入评价商品ID匹配不到商品库时，展示解析失败。
- 保存导入时如存在失败行，不允许保存。

### 5.5 权限态 / 不可操作态
- 无导入积分权限时隐藏或置灰「导入积分」入口。
- 无导入评价权限时隐藏或置灰「导入评价」入口。
- 无规则编辑权限时隐藏编辑按钮。
- 无规则删除权限时隐藏删除按钮。
- 无启停权限时状态开关置灰不可操作。
- 无评价管理权限时隐藏评价展示、设置关键词等操作。

### 5.6 状态-操作矩阵（按需）
| 状态 | 可执行操作 | 备注 |
| --- | --- | --- |
| 规则启用 | 编辑、日志、删除、停用 | 停用后规则不再参与前台拦截 |
| 规则停用 | 编辑、日志、删除、启用 | 启用后立即参与前台拦截 |

### 5.7 状态流转（按需）
- 规则新建 → 启用：新建保存后默认启用。
- 启用 → 停用：列表关闭状态开关。
- 停用 → 启用：列表打开状态开关。
- 导入未上传 → 解析成功：上传合法模板文件且所有行校验通过。
- 导入未上传 → 解析失败：上传文件表头或行数据校验失败。
- 解析成功 → 导入成功：点击保存并提交成功。
- 商品评价导入成功 → 列表新增导入评价：新增记录评价来源为导入评价。

## 6. 衍生需求与默认授权项

### 6.1 导出
- 商品屏蔽、商品限购列表本期不提供导出入口。
- 商品评价列表本期不新增导出入口。

### 6.2 日志
- 商品屏蔽 / 商品限购规则的新建、编辑、删除、启停需记录操作日志。
- 日志字段包含：日志ID、操作人、操作描述、提示信息、操作时间。

### 6.3 ID 规则
- 商品屏蔽规则 ID 建议前缀：B。
- 商品限购规则 ID 建议前缀：L。


### 6.4 端能力限制
- 前台列表、搜索、商品详情、购物车、下单链路需接入用户限购规则校验。
- 后台原型仅覆盖运营管理页面，不覆盖完整前台页面。

### 6.6 默认授权项
- 用户管理查看 / 导出 / 导入积分 / 下载模板。
- 用户限购查看。
- 商品屏蔽新建 / 编辑 / 删除 / 启停 / 日志查看。
- 商品限购新建 / 编辑 / 删除 / 启停 / 日志查看。
- 商品评价查看 / 导入评价 / 下载模板 / 评价展示 / 设置关键词。
