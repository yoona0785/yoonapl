兑换审核逻辑说明

1. 页面路径

  1. 页面路径：营销中心 - 订单兑换 - 兑换审核。
  2. 从订单兑换列表点击「兑换审核」进入。

2. 搜索逻辑

  1. 用户 ID / 手机号：关键词精准匹配。
  2. 用户昵称：关键词模糊匹配。
  3. 活动名称：下拉单选，可选所有活动，包含已删除活动。
  4. 状态：默认全部，可选待审核、通过、驳回。

3. 列表字段

  1. 提交内容：平台、店铺名称、上传截图、补充说明。
  2. 活动信息：状态、活动名称、活动 ID。
  3. 用户信息：用户昵称、手机号、用户 ID。
  4. 提交时间、审核时间均精确到秒；审核时间展示账号名称和审核提交时间。
  5. 审核状态：待审核、通过、驳回；通过展示积分，意见图片支持放大查看。

4. 审核交互

  1. 待审核状态展示「审核」操作，点击打开抽屉。
  2. 审核结果必选，无默认，可选通过、驳回。
  3. 通过时展示兑换积分，必填且只能正整数。
  4. 驳回时审核意见必填；通过时审核意见非必填，但文字或图片至少填写一项。
  5. 保存校验通过后返回审核列表并更新状态。
